# <a name="irm-protection-to-sharepoint-files"></a>对 SharePoint 文件的 IRM 保护


<p>在 SharePoint Online 中, IRM 保护应用于列表和库级别的文件。 您的组织可以使用 IRM 保护之前, 必须先设置权限管理。 IRM 依赖 azure 权限管理服务 (来自 Azure 信息保护) 来加密和分配使用限制。 一些 Office 365 计划包括 Azure 权限管理, 但并非全部。 若要了解详细信息, 请参阅<a href="https://docs.microsoft.com/azure/information-protection/understand-explore/office-apps-services-support" data-linktype="external">Office 应用程序和服务如何支持 Azure 权限管理</a>。</p> <ul> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center">Set up Information Rights Management (IRM) in SharePoint admin center</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center#irm-enable-sharepoint-document-libraries-and-lists">IRM-启用 SharePoint 文档库和列表</a></li> <li><a href="https://support.office.com/en-US/Article/Information-Rights-Management-in-Office-c7a70797-6b1e-493f-acf7-92a39b85e30c">Office 中的信息权限管理</a></li> <li><a href="https://docs.microsoft.com/en-us/office365/SecurityCompliance/information-rights-management-in-exchange-online">Exchange Online 中的信息权限管理</a></li> </ul>
