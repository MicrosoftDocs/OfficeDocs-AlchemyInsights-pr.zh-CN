# <a name="classification-labels-in-office-365"></a>Office 365 中的分类标签

使用 Office 365 中的敏感度或分类标签, 可以对敏感内容进行分类和帮助保护, 同时确保您的人员的工作效率和协作能力不阻止。

有关详细信息, 请参阅以下[文章](https://docs.microsoft.com/en-us/office365/securitycompliance/sensitivity-labels)。
