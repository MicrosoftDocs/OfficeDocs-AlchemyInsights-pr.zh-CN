# <a name="change-a-users-name-and-email-address"></a>更改用户的名称和电子邮件地址

更改用户的名称和电子邮件地址很容易 ！转到[活动用户](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users)，并选择要编辑其信息的用户。有关分步说明，您还可以使用此文章： 
  
[在 Office 365 中更改用户名和电子邮件地址](https://support.office.com/article/Change-a-user-name-and-email-address-in-Office-365-fb5ac074-e203-4e1f-9843-b9d1a3e03297?wt.mc_id=change_email_AI.aspx)
  
 **注意**： 如果您更改用户的主电子邮件地址，其旧的电子邮件地址仍将接收电子邮件 （发生这种情况自动） 有效。此外，如果您的组织在混合状态，所有电子邮件和用户的名称更改需要完成的本地 Active Directory，然后同步到云。
