# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>将用户邮箱转换为共享邮箱

将用户的邮箱转换为共享邮箱时, 所有现有的电子邮件和日历都将保留。 只有在共享邮箱中, 多个用户才能访问它, 而不是一个人。 稍后, 您可以将共享邮箱转换回用户 (专用) 邮箱。
