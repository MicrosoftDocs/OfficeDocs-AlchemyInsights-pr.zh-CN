# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>将用户邮箱转换为共享邮箱

如果您将用户邮箱转换为共享邮箱时，将保留所有现有的电子邮件和日历。仅现在很多人将能够访问而不是一个人共享邮箱中。以后，您可以将共享的邮箱转换回用户 （专用） 邮箱。
