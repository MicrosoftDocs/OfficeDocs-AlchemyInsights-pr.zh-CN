# <a name="onedrive-puid-mismatch"></a>OneDrive PUID 不匹配

您似乎在 OneDrive for Business 中遇到 [ID 不匹配](https://docs.microsoft.com/sharepoint/troubleshoot/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site) 问题。 请按照下面的步骤操作，以便我们能够尽快为你提供帮助：

1. 单击 "  **联系人支持**"，然后单击 "  **新建服务请求**"。
2. 将标题和说明保留为  **OneDrive 网站用户 ID 不匹配**。
3. 提交票证后，请使用以下信息对其进行更新：
    - 受影响的 OneDrive 网站 URL。
    - 受影响的用户 (UPN) 的用户主体名称。
