# <a name="onedrive-puid-mismatch"></a>OneDrive PUID 不匹配

似乎你在 OneDrive for Business 中遇到 [ID](https://docs.microsoft.com/sharepoint/troubleshoot/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site) 不匹配。 若要允许我们尽早为您提供帮助，请执行以下步骤：

1. 单击 **"联系支持** 人员"，然后单击 **"新建服务请求"。**
2. 将标题和说明保留为  **OneDrive 网站用户 ID 不匹配**。
3. 提交票证后，使用以下信息更新它：
    - 受影响的 OneDrive 网站 URL。
    - 受影响用户 (UPN) 名称。
