# <a name="prepaid-renewal"></a>预付续订

1. 在管理中心，转到[订阅](https://go.microsoft.com/fwlink/p/?linkid=842054)页面，或选择“**帐单**”\>“**订阅**”。
    
2. 选择要扩展的预付订阅, 然后选择 "**扩展**"。
    
3. 选择 "**产品密钥**" 选项。 (如果这是在零售位置购买的产品密钥, 则可以选择使用其他付款方式进行扩展。)
    
4. 输入产品密钥。
    
5. 请确保信息正确, 然后选择 "**下一步**"。
    
6. 选择单选按钮以延长现有许可证的结束日期, 然后选择 "**下一步**"。
