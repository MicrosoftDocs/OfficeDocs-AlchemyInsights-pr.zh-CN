# <a name="calendar-publishing-information"></a>日历发布的信息

- 管理员可以启用或禁用日历发布共享和日历共享的组织。有关详细信息，请参阅[启用日历发布使人们可以共享信息](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)和[共享与外部用户的日历](https://support.office.com/article/FB00DD4E-2D5F-4E8D-8FF4-94B2CF002BDD)。
    
- Outlook 用户可以共享他们自己的日历。有关详细信息，请参阅[共享与其他人的 Outlook 日历](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)。
