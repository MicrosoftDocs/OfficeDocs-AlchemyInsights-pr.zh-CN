# <a name="stream-noise-suppression"></a>流媒体降噪

视频所有者启用降噪功能后，观看者可在播放期间通过它将讲话声与其他声音和音乐隔开。 可对每个新上传的视频启用或禁用降噪功能。 有关详细信息，请参阅[降噪](https://docs.microsoft.com/stream/noise-suppression)。
