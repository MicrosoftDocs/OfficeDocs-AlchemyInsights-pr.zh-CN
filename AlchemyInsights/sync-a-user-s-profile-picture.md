# <a name="sync-a-users-profile-picture"></a>同步用户配置文件图片

SharePoint Online 中的配置文件图片同步需要启动其照片同步的用户的操作。没有能自动同步到 SharePoint Online 的照片。SharePoint Online 仅同步来自 Exchange Online，Azure Active Directory 照片。
  
如果用户已从不执行的操作的提示同步或他们已更新其配置文件图片，并在新的其中一个不出现在 SharePoint Online，用户可能需要提示以便同步。下面是如何才能此：
  
1. 打开 web 浏览器中，输入 Private 或 Incognito 会话并[转到 Office.com](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.office.com%2F&amp;data=02%7C01%7C%7Cbad62c504a36446096e108d614dec653%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C636719344369977864&amp;sdata=Km7ZnN8FHSouZbxOiEpQAGIKsK82SHr25uYCh3Gc%2F3s%3D&amp;reserved=0)。
  
2. 使用其工作或学校的帐户，登录，然后单击 OneDrive 或 SharePoint 图块。
