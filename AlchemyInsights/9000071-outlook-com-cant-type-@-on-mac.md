# <a name="how-to-type-special-characters-on-a-mac"></a>如何在 Mac 上键入特殊字符

若要在 Mac 上键入特殊字符, 请使用键盘快捷方式⌘**命令** + **控制** + **空间**打开 "**特殊字符**" 菜单。

@ 的快速快捷方式也是**ALT** + **64**。
