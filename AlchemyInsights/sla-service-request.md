# <a name="credit-request-due-to-a-service-outage"></a>由于服务中断导致的信用请求

若要应用于由于服务中断导致的信用请求，请打开服务请求。

请务必在 Microsoft 365 管理中心的 "[服务运行状况" 页](https://docs.microsoft.com/office365/enterprise/view-service-health)中找到事件 ID。

请注意，在处理你的请求时，它会经历验证以观察中断的影响。
