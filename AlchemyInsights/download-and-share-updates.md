# <a name="download-and-share-updates"></a>下载和共享更新

“传递优化”下载模式可实现多台电脑之间的共享，而不是逐个下载更新。 有关详细信息，请参阅[适用于 Windows 10 更新的传递优化](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization)。  

可用的下载模式包括：  
- 仅限 HTTP  
- LAN  
- 组  
- Internet  
- 简单  
- 旁路
