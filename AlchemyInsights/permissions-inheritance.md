# <a name="how-permissions-inheritance-works-in-sharepoint"></a>权限继承在 SharePoint 中的工作方式

默认情况下，SharePoint 中的权限从层次结构中的较高版本继承。 因此，文件从文件夹继承其权限，这将从库继承其权限，这会从网站继承其权限，这将从网站集继承权限。
  
有关删除唯一权限和还原继承的信息，请参阅[编辑和管理列表或库的权限](https://go.microsoft.com/fwlink/?linkid=869946)。
