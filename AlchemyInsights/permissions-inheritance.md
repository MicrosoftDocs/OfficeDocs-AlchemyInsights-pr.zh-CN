# <a name="how-permissions-inheritance-works-in-sharepoint"></a>权限继承在 SharePoint 中的工作方式

默认情况下，在 SharePoint 中的权限继承从较高层次结构中。因此，文件从其权限继承其权限继承的网站，其权限继承的网站集库文件夹继承其权限。
  
有关删除独有权限和还原继承的信息，请参阅[编辑和管理列表或库的权限](https://go.microsoft.com/fwlink/?linkid=869946)。
