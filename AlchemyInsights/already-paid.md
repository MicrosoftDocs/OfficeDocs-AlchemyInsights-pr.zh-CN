# <a name="already-paid"></a>已支付？

如果你已发送付款, 你可以让我们知道:
  
1. 转到 "[订阅](https://go.microsoft.com/fwlink/p/?linkid=842054)" 页面, 或选择 "**帐单** \> **订阅**"。
    
2. 选择一个订阅, 并在 "**我的帐单**" 部分, 选择 "**查看帐单**"。
    
3. 在 "**帐单详细信息**" 页上, 选择 **"已支付"？** 系统将要求你输入与发送的付款相关的信息。 
    
4. 输入所有必需的信息后, 选择 "**发送**"。
