# <a name="already-paid"></a>已付费？

如果已发送付款，可以通知我们：
  
1. 在管理中心，转到“**计费**”\>“[订阅](https://go.microsoft.com/fwlink/p/?linkid=842054)”页面。

2. 选择订阅，在"我的帐单"**部分**，选择"**查看帐单"。**

3. 在" **帐单详细信息"** 页面上，选择" **已付费"？** 将要求您输入与已发送付款有关的信息。

4. 输入所有必需信息后，选择"发送 **"。**
