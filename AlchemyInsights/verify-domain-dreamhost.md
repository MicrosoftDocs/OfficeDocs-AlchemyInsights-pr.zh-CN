# <a name="verify-your-domain-with-dreamhost"></a>使用 Dreamhost 验证域

若要验证域，您需要向域注册机构添加 TXT 记录，以验证您是否拥有您的域。 

首先，通过选择"设置 [""域](https://admin.microsoft.com/Adminportal#/Domains)"， \> 然后"添加域"，按照 **"添加域"向导操作**。
  
有关 [分步说明，请参阅在 Dreamhost for Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-dreamhost) 创建 DNS 记录。
