# <a name="move-a-phone-number-from-another-provider-to-microsoft"></a>将电话号码从另一个提供商移动到 Microsoft

是否需要将其他提供商的电话号码移动到 Microsoft？ 可以在[管理组织的电话号码](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization)中找到不同国家和地区的电话号码。 选择你所在的国家或地区以查找有关转移号码的信息。
