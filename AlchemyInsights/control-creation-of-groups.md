# <a name="control-creation-of-microsoft-365-groups"></a>控制 Microsoft 365 组的创建

默认情况下，公司内的所有用户都能够创建组。 如果需要，你可以更改此权限，并将创建组的权利分配给公司内的一组人员。 请注意，这将影响从任何 Microsoft 365 服务组织（如 Outlook、Teams、Planner 等 (）创建) 。
