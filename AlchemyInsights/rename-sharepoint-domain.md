# <a name="rename-your-sharepoint-domain"></a>重命名您的SharePoint域

目前，有一个专用预览版，用于重命名你的SharePoint域。

有关重命名过程和限制的信息，请访问 [https://aka.ms/SPOTenantRename](https://aka.ms/SPOTenantRename) 。

如果你想要将自己指定到私人预览版中，可以在 中这样做 [https://aka.ms/SPOTenantRenameNomination](https://aka.ms/SPOTenantRenameNomination) 。 请注意，这一名单不一定意味着你将被接受。 我们将根据情况查看这些联系人，并且仅在你被接受时联系。
