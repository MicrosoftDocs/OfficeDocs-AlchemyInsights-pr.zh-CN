# <a name="focused-inbox-in-outlook"></a>Outlook 中的重点收件箱

重点收件箱将收件箱分隔为两个选项卡—重点和其他。 最重要的邮件位于 "重点" 选项卡上，而 rest 仍可在 "其他" 选项卡上轻松访问（但不是这样）。
