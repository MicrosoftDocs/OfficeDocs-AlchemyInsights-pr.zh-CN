# <a name="cant-add-default-2010-approval-workflow"></a>无法添加默认 2010年审批工作流

Microsoft SharePoint 网站集，您不能对列表或库添加全局可重用工作流 （如"审批-SharePoint 2010"）。审批工作流可能会丢失。若要解决此问题请参阅[无法添加到 SharePoint 列表或库的全局可重用工作流](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)。
