# <a name="cant-start-or-join-a-skype-meeting-broadcast"></a>无法启动或加入 Skype 会议直播？

如果您在启动或加入 Skype 会议直播时遇到问题，作为事件团队成员（会议制造者），请务必仔细检查您的配置。 Skype 会议直播需要手动设置一些步骤。 按照本文中的步骤操作，以确保正确配置了网络和 Skype for business Online 门户，并且受影响的用户未受到冲突许可证的阻止：

[为 Skype 会议直播设置网络](https://docs.microsoft.com/SkypeForBusiness/set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast)
