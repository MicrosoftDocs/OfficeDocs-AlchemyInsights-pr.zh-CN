# <a name="view-reports-on-onedrive-and-sharepoint-activity-and-usage"></a>查看有关 OneDrive 和 SharePoint 的活动和使用情况报告

转到[Office 365 管理中心中的使用率报告页面](https://admin.microsoft.com/AdminPortal/Home)。在**选择报表**，下选择 OneDrive 或 SharePoint 的活动或使用率报告。
