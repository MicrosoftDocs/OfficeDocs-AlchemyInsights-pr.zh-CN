# <a name="view-reports-on-onedrive-and-sharepoint-activity-and-usage"></a>查看 OneDrive 和 SharePoint 活动和使用情况的报告

转到 [Microsoft 365 管理中心中的 "使用率报告" 页](https://admin.microsoft.com/AdminPortal/Home)。 在 " **选择报告**" 下，选择 "OneDrive" 或 "SharePoint" 活动或使用率报告。
  
[有关 OneDrive 报告的详细信息](https://go.microsoft.com/fwlink/?linkid=875239)
  
[有关 SharePoint 报告的详细信息](https://go.microsoft.com/fwlink/?linkid=875240)
