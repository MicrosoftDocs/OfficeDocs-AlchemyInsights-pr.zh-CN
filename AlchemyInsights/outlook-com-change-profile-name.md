# <a name="change-your-profile-name-in-outlookcom"></a>在 Outlook.com 中更改您的配置文件名称

1. 转到[你的 Microsoft 帐户信息](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 选择 "**编辑名称**"。
3. 输入名字和姓氏, 然后选择 "**保存**"。

请允许长达24小时对你的配置文件名称所做的更改生效。
