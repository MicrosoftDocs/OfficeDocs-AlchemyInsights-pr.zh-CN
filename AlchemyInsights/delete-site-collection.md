# <a name="delete-a-sharepoint-site"></a>删除 SharePoint 网站

1. 在新的 SharePoint 管理中心中, 在 "网站" 页上, 单击要删除的网站旁边的复选框。
    
2. 在功能区工具栏上, 单击 "**删除**"。
