# <a name="delete-a-sharepoint-site-collection"></a>删除 SharePoint 网站集

1. 在经典 SharePoint 管理中心的 "网站集" 页上, 选中网站集旁边的复选框。
    
2. 在 "网站集" 选项卡上, 选择 "**删除**"。
