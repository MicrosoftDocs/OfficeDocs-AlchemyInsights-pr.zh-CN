# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>使用新版 OneDrive 同步客户端同步 SharePoint 文件

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">“使用资源管理器打开”命令将打开一个本地 Windows 资源管理器实例，其中显示托管 SharePoint 网站的服务器上的文件夹结构。这就是说，我们建议<a href="https://support.office.com/zh-CN/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">将 SharePoint 文件与提供<a href="https://support.office.com/zh-CN/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">文件随选</a>的新 OneDrive 同步客户端</a>同步，因为它可以提供文件的本地访问并提供最佳性能</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">如果已选择使用资源管理器视图而不是使用新同步客户端，请确保按照下文中所述的步骤和最佳做法。</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/zh-CN/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">如何使用"使用资源管理器打开"命令对 SharePoint Online 中的问题进行故障排除</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/zh-CN/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">通过“使用资源管理器打开”命令来复制或移动库文件</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">注意：</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">在新的库体验中，“使用资源管理器打开”按钮不会显示。单击右上角的“<strong>查看</strong>”下拉菜单（下拉菜单的名称随当前视图的更改而变化），然后在“文件资源管理器”中单击“<strong>查看</strong>”。</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">注意：</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> SharePoint“使用资源管理器打开”按钮使用 <strong>ActiveX 控件</strong>，因此仅在 Internet Explorer 10 或 11 中受支持。“使用资源管理器打开”不适用于采用 Microsoft Edge、Google Chrome、Mozilla Firefox 的 Windows 或 Mac 平台。出于此原因，“&ldquo;资源管理器视图&rdquo;”选项可能会灰显。</span></p> <ul> <li>
