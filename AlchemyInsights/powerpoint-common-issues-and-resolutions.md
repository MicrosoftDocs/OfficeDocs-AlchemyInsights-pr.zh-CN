# <a name="powerpoint-common-issues-and-resolutions"></a>PowerPoint 常见问题和解决方法

下面是 PowerPoint 的一些经常请求的主题：

1. [PowerPoint 没有响应、挂起或冻结](https://support.office.com/article/PowerPoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d)。

2. [PowerPoint 无法从所选文件插入视频](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c)。

3. **视频**：[从你的计算机插入视频文件](https://support.office.com/article/insert-and-play-a-video-file-from-your-computer-f3fcbd3e-5f86-4320-8aea-31bff480ed02)、[从 YouTube 插入视频](https://support.office.com/article/Insert-a-video-from-YouTube-or-another-site-8340ec69-4cee-4fe1-ab96-4849154bc6db)、[在 PowerPoint 中支持的视频和音频文件格式](https://support.office.com/article/video-and-audio-file-formats-supported-in-powerpoint-d8b12450-26db-4c7b-a5c1-593d3418fb59)。 

4. **入门：了解**如何在[开始使用针对 web 的 PowerPoint](https://support.office.com/article/Get-started-with-PowerPoint-for-the-web-6303da7a-402a-4300-8b1c-160e8940cc34)时创建、保存、编辑和共享演示文稿。

5. **打印**：有关与打印相关的最佳实践和问题，请查看[打印您的 PowerPoint 幻灯片和备注](https://support.office.com/article/Print-your-PowerPoint-slides-handouts-or-notes-194d4320-aa03-478b-9300-df25f0d15dc4)中的信息。 

6. **字体**：若要与演示文稿共享字体，请阅读[Word 或 PowerPoint 中的嵌入字体](https://support.office.com/article/Embed-fonts-in-Word-or-PowerPoint-cb3982aa-ea76-4323-b008-86670f222dbc)。

7. **功能**：在 Windows、Mac、web 和移动平台上[比较 PowerPoint 功能](https://support.office.com/article/Compare-PowerPoint-features-on-different-platforms-90986850-227c-4b25-938e-1c5838166b8b#bm11)。
