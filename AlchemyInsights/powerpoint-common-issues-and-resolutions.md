# <a name="powerpoint-common-issues-and-resolutions"></a>PowerPoint 常见问题和解决方法

下面是 PowerPoint 的一些经常请求的主题：

1. **在 Microsoft 团队中使用 PowerPoint**：在 [团队中的会议中共享 powerpoint 幻灯片](https://support.microsoft.com/office/share-content-in-a-meeting-in-teams-fcc2bf59-aecd-4481-8f99-ce55dd836ce8#ID0EABAAA=Desktop) ，或在 microsoft 团队中协作处理文件。
1. [PowerPoint 没有响应、挂起或冻结](https://support.office.com/article/PowerPoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d)。
1. **基于 web 的 powerpoint**： [特定功能在基于 web 的 powerpoint 中的行为方式。](https://support.microsoft.com/office/how-certain-features-behave-in-web-based-powerpoint-a931f0c8-1305-4428-8f7c-9cfa00ef28c5)
1. [PowerPoint 无法从所选文件插入视频](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c)。
1. **打印**：有关与打印相关的最佳实践和问题，请查看 [打印您的 PowerPoint 幻灯片和备注](https://support.office.com/article/Print-your-PowerPoint-slides-handouts-or-notes-194d4320-aa03-478b-9300-df25f0d15dc4)中的信息。 
1. **功能**：在 Windows、Mac、web 和移动平台上 [比较 PowerPoint 功能](https://support.office.com/article/Compare-PowerPoint-features-on-different-platforms-90986850-227c-4b25-938e-1c5838166b8b#bm11) 。
