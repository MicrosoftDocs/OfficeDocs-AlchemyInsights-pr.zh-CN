# <a name="migrate-files-from-dropbox"></a>从 Dropbox 迁移文件

如果要从 Dropbox 迁移到 SharePoint Online，可以考虑使用 **Mover**。

有关详细信息，请参阅 [授权 Dropbox 连接器](https://docs.microsoft.com/sharepointmigration/mover-dropbox)。
