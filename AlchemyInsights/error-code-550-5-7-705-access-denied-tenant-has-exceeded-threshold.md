# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 访问被拒绝，租户超过阈值

已通过 Office 365 组织发送太多垃圾邮件或批量邮件时，将出现此消息。若要删除此块，您需要了解和发送给支持代理，解释原因，以及纠正基础的问题。
