# <a name="sharepoint-migration-with-spmt"></a>使用 SPMT 进行 SharePoint 迁移

SharePoint 迁移工具旨在用于从最小文件集到大规模企业迁移的迁移，它将允许你将信息转移到云，并利用 Microsoft 365 中最新的协作、智能和安全解决方案。
