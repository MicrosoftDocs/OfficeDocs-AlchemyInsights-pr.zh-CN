# <a name="manage-a-group-without-an-exchange-license"></a>管理不带 Exchange 许可证的组

如果你的管理员帐户没有邮箱，则在尝试编辑 Microsoft 365 组时，可能会收到有关缺少产品许可证的错误。
  
由于你是租户管理员，因此可以编辑组，即使你没有 Exchange 许可证也是如此。 只需转到 [Exchange 管理中心](https://outlook.office365.com/ecp.aspx) \> **组** 即可管理 Microsoft 365 组。
  
如果要编辑管理中心中的组，请将 Exchange 许可证添加到管理员帐户。 您需要等到邮箱完全创建后，才能编辑组。
