# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>更改 SharePoint 或 OneDrive 中的显示语言设置 

您无法更改 SharePoint 或 OneDrive 的默认语言设置，因为这些设置是在创建 Microsoft 365 订阅时定义的。 不过，您可以更改显示语言设置。

若要更改你的个人设置，请参阅[更改你的个人语言和区域设置](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)。

若要更改网站的用户界面，请参阅[选择要对网站的用户界面提供的语言](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)。

有关排除语言设置故障的详细信息，请参阅[不能更改 SharePoint Online 网站集的默认语言](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)。
