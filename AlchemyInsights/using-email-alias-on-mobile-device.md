# <a name="email-aliases-on-mobile-devices"></a>移动设备上的电子邮件别名

电子邮件别名只是在邮箱上配置的另一个电子邮件地址，因此发送给用户别名的邮件将被传递到其邮箱，并可在其移动设备上阅读。 不需要其他配置。

但是，用户不能答复邮件，也不能使用别名作为发件人地址来发送新邮件。 所有传出邮件都将来自其主电子邮件地址。
