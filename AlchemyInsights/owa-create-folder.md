# <a name="create-a-folder"></a>创建文件夹

1. 在 Outlook 网页的文件夹窗格的底部，选择 "**新建文件夹**"。 您可能需要在文件夹窗格中向下滚动才能看到 "**新建文件夹**" 按钮。
1. 在文本框中，输入文件夹的名称，然后按 Enter。

若要了解详细信息，请参阅[在 web 上使用 Outlook 中的邮件文件夹](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f)。
