# <a name="learn-how-to-configure-agenda-settings"></a>了解如何配置日程安排设置

"议程设置" 位于 "Outlook 网页" 中的 "**日历**设置" 中。 有关详细信息, 请参阅[web 上的 Outlook 中的日历设置](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f)。

管理员可以使用 Exchange Online PowerShell 中的**get-calendarnotification** cmdlet 配置议程设置。 有关详细信息, 请参阅[get-calendarnotification](https://technet.microsoft.com/library/dd351284)。
