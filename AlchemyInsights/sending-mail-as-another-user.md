# <a name="sending-mail-as-another-user"></a>以其他用户的形式发送邮件

在*user1* *能够以用户2身份发送*电子邮件之前，需要在用户2的*邮箱上授予对用户* *1*的权限。 若要了解如何执行此操作，请参阅向另一个用户授予邮箱权限。

授予权限后 (此操作可能需要大约30分钟才能生效) - *User1* 将按照创建或回复电子邮件中的其他人所述的步骤，以 *用户身份为* 用户2进行发送。
