# <a name="create-a-folder"></a>创建一个文件夹

1. 在左窗格中，选择**创建新文件夹**![创建新文件夹按钮](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png)旁边**文件夹**。 
    
2. 在新建文件夹框中，输入文件夹的名称，然后按 Enter。
    
若要了解详细信息，请参阅[Create Outlook.com 中的文件夹](https://go.microsoft.com/fwlink/p/?linkid=873114)。
