# <a name="change-font-or-font-size"></a>更改字体或字号

阅读电子邮件时, 可以使用浏览器的缩放功能放大或缩小屏幕上的文本。
  
撰写邮件时更改字体或字号:
  
1. 在撰写窗格的底部, 选择 "**字体**" 或 "**字号**" 图标。
2. 选择要使用的字体或字号。

如果在撰写窗格底部看不到 "格式选项" 菜单, 请选择 "**格式设置选项**" 图标。
  
有关详细信息, 请参阅[更改 Outlook.com 中电子邮件的字体大小和外观](https://support.office.com/article/0b4eb323-23fc-4d5d-adbf-cae14c9c0386?wt.mc_id=Office_Outlook_com_Alchemy)。
