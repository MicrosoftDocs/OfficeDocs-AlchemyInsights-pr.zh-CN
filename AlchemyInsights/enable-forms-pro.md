# <a name="enable-or-disable-forms-pro"></a>启用或禁用 Forms Pro

如果要尝试启用 Microsoft Forms Pro，它将作为[Dynamics 365 企业版](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)许可证的一部分包含在内。 此外，管理员还可以购买[Microsoft Forms Pro 许可证](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant)。 您还可以从用户中[删除 Microsoft Forms Pro 许可证](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1)。
