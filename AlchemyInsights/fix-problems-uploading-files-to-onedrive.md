# <a name="fix-problems-uploading-files-to-onedrive"></a>修复问题将文件上载到 OneDrive

如果文件不上载，尝试此：
  
- 重命名该文件，然后重试。不允许一些文件名和文件名称中的字符。 
    
- 如果您正在同步该文件，将其移出您 OneDrive 文件夹 （例如，将其移动到您的桌面），以及当您的 OneDrive 同步的状态显示为"最新的"将其移回。
