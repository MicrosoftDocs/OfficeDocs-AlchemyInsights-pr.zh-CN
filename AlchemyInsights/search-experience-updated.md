# <a name="search-experience-updated"></a>更新了搜索体验

搜索现在位于应用程序的底部。 点击此项可直接访问你的主要联系人、即将到来的旅行路线、包交付和最近的附件。 您可以从一个搜索框中搜索任何帐户中的电子邮件、文件和联系人，或使用我们的新筛选器来缩小搜索范围。
