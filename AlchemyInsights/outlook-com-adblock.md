# <a name="adblock"></a>AdBlock

某些用户在使用 Google Chrome 和 AdBlock 扩展时, 在阅读或撰写电子邮件时, 当前遇到问题。 若要解决此问题, 请执行以下操作:

1. 使用 [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) 或其他 web 浏览器。
1. 从 AdBlock 设置中排除 Outlook.com。在 Google Chrome 的右上角, 选择 " **AdBlock** ", 然后选择 " **不在此域上的页面上运行"**。刷新浏览器以再次加载 Outlook.com。
