# <a name="manage-groups-in-yammer"></a>在 Yammer 中管理组

- **组创建权限。** Yammer 不控制谁可以创建组。 但是，可以管理已连接的 Microsoft 365 组的组创建。 有关详细信息，请参阅[管理可创建 Microsoft 365 组的用户](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)。

- **动态组成员身份。** 若要了解如何为已连接 Microsoft 365 组的 Yammer 创建动态成员资格，请参阅[在 Yammer 中创建动态组](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group)。

- **还原已删除的组。** 可以通过[还原已删除的 Microsoft 365 组](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group)还原 Yammer 组。删除后，不能还原未连接的组。

有关管理组的详细信息，请参阅[在 Yammer 中管理组](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2)。 有关连接的 Microsoft 365 组的信息，请参阅[Yammer 和 Microsoft 365 组](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups)。
