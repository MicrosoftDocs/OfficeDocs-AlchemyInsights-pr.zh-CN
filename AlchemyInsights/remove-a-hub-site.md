# <a name="remove-a-hub-site"></a>删除中心网站

在新 SharePoint 管理中心中注销中心网站
  
1. 在新的 SharePoint 管理中心中, 在 "**网站**" 下, 单击 "**活动网站**"。

2. 选择网站, 单击 "**中心网站**", 然后单击 "**取消注册为中心网站**"。

3. 单击“确定”****。

[了解有关中心网站的详细信息](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
