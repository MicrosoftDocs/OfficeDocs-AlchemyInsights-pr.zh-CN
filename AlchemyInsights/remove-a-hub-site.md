# <a name="remove-a-hub-site"></a>删除中心站点

取消注册新的 SharePoint 管理中心中的中心网站
  
1. 在新的 SharePoint 管理中心，在**网站**下单击**活动网站**。 
    
2. 选择的站点单击**中心站点**，然后单击**注销作为中心站点**。 
    
3. 单击“**确定**”。 
    
[了解有关中心站点](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
