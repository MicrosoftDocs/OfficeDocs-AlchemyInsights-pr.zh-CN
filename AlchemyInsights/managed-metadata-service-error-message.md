# <a name="managed-metadata-service-error-message"></a>托管元数据服务错误消息

打开术语库时，您可能会看到一条错误消息:"的托管元数据服务或连接当前不可用。应用程序池或 Managed Metadata Web Service 可能尚未启动。请请与管理员联系。"
  
如果超过 24 小时过去后第一次看到此问题，请登录支持案例。我们了解 24 小时可能看起来像很长时间。在许多情况下，我们已经在处理解决方案因此请向我们提供至少 24 小时。
