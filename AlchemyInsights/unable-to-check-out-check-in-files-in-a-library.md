# <a name="unable-to-check-out-or-check-in-files-in-a-library"></a>无法签出或签入库中的文件

- 如果库中有必填字段，且该字段未填充值，则文件会呈签出状态，直至其被填充时才会签入。

- 如果在文件标题中有特殊字符，可能会发生问题。 删除特殊字符并尝试签入文件。

- 如果用户签出文件，文件会被锁定。  签出文件人员不在时，库的所有者可以从 **库设置->管理文件** 签入文件。
