# <a name="set-up-microsoft-information-protection"></a>设置 Microsoft 信息保护

概括了解可应用于信息保护策略的功能，以便你确信敏感信息得到了保护。 使用四个阶段的生命周期方法来发现、分类、保护和监视敏感信息。 “Microsoft 信息保护设置”指南提供了有关完成每个阶段的指导。
