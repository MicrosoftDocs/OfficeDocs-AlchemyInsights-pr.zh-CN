# <a name="block-ads-in-microsoft-edge-on-an-adroid-or-ios-device"></a>在 Adroid 或 iOS 设备上阻止 Microsoft Edge 中的广告

若要阻止除 AdBlock 可接受的所有广告之外的所有广告，请执行以下操作：
1. 选择 **...** > **设置**  > **内容阻止**。
2. 启用块状广告。

阻止所有广告：
1. 选择 **...** > **设置**  > **内容阻止**。
2. 启用块状广告。
3. 选择 " **高级设置**"。
4. 禁用 "允许可接受的广告"。
