# <a name="access-denied-banned-sender"></a>访问被拒绝，禁止发件人

 **错误**： 从 RCPT 命令，承载用于远程服务器的 SMTP 错误：*格式*-com.mail.protection.outlook.com ( *xxx.xxx.xxx.xxx* ) 原因： 550 5.7.511 访问被拒绝，禁止发件人 [ *xx.xxx.x.xxx* ]。 
  
如果向 Office 365 收件人发送电子邮件时遇到此错误，则会将受影响的帐户移除的阻止发件人列表中， [https://sender.office.com](https://sender.office.com)。
