# <a name="disable-excel-live-preview"></a>禁用 Excel Live Preview

如果您在 Excel 中收到一条错误消息, 指出 "剪贴板存在问题, 但仍可以在此工作簿中粘贴内容", 请按照以下步骤操作, 尝试禁用实时预览功能:

1. 在 Excel 中, 单击 "**文件** > **选项**"。
3. 在 " **Excel 选项**" 对话框中, 选择 "**常规**"。
4. 在 "**用户界面选项**" 下, 清除 "**启用实时预览**"。
5. 单击“确定”****。
6. 重新启动 Excel。
