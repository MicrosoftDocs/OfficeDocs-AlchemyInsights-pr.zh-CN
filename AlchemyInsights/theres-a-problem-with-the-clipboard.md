# <a name="resolving-excel-clipboard-error"></a>解决 Excel 剪贴板错误

如果在 Excel 中收到一个错误，指出"剪贴板有问题，但仍可以在此工作簿中粘贴内容"，请尝试按照以下步骤禁用"实时预览"功能：

1. 在 Excel 中，单击 **"文件**  >  **选项"。**
3. 在 **"Excel 选项"** 对话框中，选择"常规 **"。**
4. 在 **"用户界面选项"下**，清除"**启用实时预览"。**
5. 单击“**确定**”。
6. 重新启动 Excel。
