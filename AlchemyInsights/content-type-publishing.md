# <a name="content-type-publishing"></a>内容类型发布

如果您希望所有用户都能在组织中使用内容类型，则最好在所有网站集中进行发布。 若要发布内容类型，请参阅 [从内容发布中心发布内容类型](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02)。
