# <a name="create-a-shared-mailbox"></a>创建共享邮箱

共享邮箱允许多个人发送和接收别名电子邮件 (如**sales@contoso.com**), 而无需分配另一个许可证。
  
有关详细信息, 请参阅[创建共享邮箱](https://support.office.com/client/871a246d-3acd-4bba-948e-5de8be0544c9)。
