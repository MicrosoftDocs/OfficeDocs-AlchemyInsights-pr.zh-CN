# <a name="restore-a-microsoft-365-group-using-the-exchange-admin-center"></a>使用 Exchange 管理中心还原 Microsoft 365 组

1. 在 Microsoft 365 管理中心的左侧导航中，展开"**管理** 中心"，然后选择 **"Exchange"。**
    
2. 在 Exchange 管理中心，选择" **收件人**"，然后选择" **组**"。你可以查看组是处于活动还是软删除状态。如果该组已被永久删除，则不会将其列出。
    
3. 若要查看组被软删除的确切时间，请选择该组并在右侧窗格中查看信息。
    
4. 选择要还原的 Microsoft 365 组，然后选择还原图标。
    
5. 选择刷新 !['刷新'图标](media/6464df90-2a91-4c1f-92a6-9a38c7696ac3.gif) 以更新页面上的信息。组将显示为活动状态。
