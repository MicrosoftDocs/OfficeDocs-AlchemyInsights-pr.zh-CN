# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Outlook 移动设置共享邮箱的疑难解答

从移动设备安装的 Outlook 应用程序，您无法访问共享的邮箱。此功能吗？让 Outlook 工程团队知道。[此处投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
您可以仅使用共享的邮箱移动设备上打开的浏览器中，登录到 Office 365，然后转 web 上的 Outlook。有关说明，请参阅[Add 中 Outlook web 上共享邮箱](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)。从 Outlook web 上您将能够访问共享的邮箱。
  
了解详细信息：[获取为 Outlook for iOS 和 Android 应用程序内帮助](https://support.office.com/en-us/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
