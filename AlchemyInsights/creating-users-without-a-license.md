# <a name="create-mailboxes-that-dont-require-licenses"></a>创建不需要许可证的邮箱

- 如果需要多人的组电子邮件地址，请参阅 [创建共享邮箱](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)。

- 若要允许团队成员协同工作并共享电子邮件和文件，请参阅 [了解 Microsoft 365 组](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)。
