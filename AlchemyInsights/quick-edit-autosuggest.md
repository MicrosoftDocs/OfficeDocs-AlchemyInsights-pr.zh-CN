# <a name="quick-edit-autosuggest"></a>快速编辑 AutoSuggest

可通过两种方法编辑 SharePoint 列表中的项目。 可以在列表视图中编辑单个项，这是默认方法。 或者，为了提高工作效率，也可以在快速编辑视图中编辑多个项目。 需要“参与”级别权限或更高权限才能编辑项目。 有关详细信息，请参阅 [在 SharePoint Online 中编辑列表项](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7)。
