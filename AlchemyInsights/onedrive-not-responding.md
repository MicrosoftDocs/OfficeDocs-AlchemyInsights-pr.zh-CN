# <a name="onedrive-not-responding"></a>OneDrive 不响应

对于新创建的帐户，用户需要使用其自己的帐户登录到 Microsoft 365 管理中心，然后单击 "OneDrive" 以手动启动 OneDrive 网站的预配。
  
但是，如果用户已单击 OneDrive，且其个人网站未在24小时内设置，请记录支持票证以获取进一步帮助。
