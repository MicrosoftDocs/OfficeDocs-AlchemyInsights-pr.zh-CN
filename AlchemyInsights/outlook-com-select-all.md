# <a name="how-to-select-all-in-outlookcom"></a>如何在 Outlook.com 中 "全选"

如果所有邮件都可见, 则只能选择 "全部"。 这是为了防止对无法看到的邮件执行意外操作。

**按住 CTRL 并单击**选择多个单个邮件。

**按住 SHIFT 并单击**选择您选择的第一条和最后一封邮件之间的所有邮件。
