# <a name="get-access-to-microsoft-bookings"></a>获取 Microsoft Bookings 访问权限

默认情况下，Microsoft Booking 适用于 **Microsoft 365 教育版 A3、A5** 和 **Microsoft 365 商业标准版（以前称为“商业高级版”）** 和 **Microsoft 365 商业高级版（以前称为“商业版”）**。 Bookings 作为一项附加产品提供给**企业版 E3 和 E5 客户**，有效期至 2020 年 5 月 12 日。 默认情况下将启用该功能。

在 **Microsoft 365 管理中心**启用 Microsoft Bookings。

1. 转到“设置”>[“设置”](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/Settings/Services)****，然后单击“Bookings”****。
2. 单击“允许组织使用 Bookings”**** 复选框，为组织启用或禁用预订。
3. 单击“保存更改”****。

有关详细信息，请参阅[获取对 Microsoft Bookings 的访问权限](https://support.microsoft.com/zh-CN/office/get-access-to-microsoft-bookings-5382dc07-aaa5-45c9-8767-502333b214ce)。
