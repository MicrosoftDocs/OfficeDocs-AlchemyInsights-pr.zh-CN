# <a name="delete-a-sharepoint-team-site"></a>删除 SharePoint 团队网站

1. 转到网站。
    
2. 单击右上角的 "**设置**" 图标。 
    
3. 单击 "**网站**设置"。 在 "**网站信息**或**网站操作**" 下, 单击 "**删除此网站**", 然后单击 "**删除**"。
