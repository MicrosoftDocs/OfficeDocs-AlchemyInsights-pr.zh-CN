# <a name="access-services"></a>Access Services

SharePoint Online 中的 Access Services 功能已被停用。不再可以创建新的基于 Access 的 web 应用程序和 web 数据库，并且现有 web 应用程序和 web 数据库将关闭按年 4 月，2018年。Access 桌面数据库 (.accdb) 不受此更改。我们建议作为的替代方式访问 Microsoft PowerApps web 应用程序和 web 数据库。 
  
[了解 Microsoft PowerApps](https://powerapps.microsoft.com/)
