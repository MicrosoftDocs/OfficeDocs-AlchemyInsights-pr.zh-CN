# <a name="download-microsoft-stream-videos"></a>下载 Microsoft Stream 视频

**如果你上传了视频文件**或者你是**视频的所有者**，则可以**下载**最初上传的视频。

1. 在 Stream 中，找到要下载的视频。

2. 单击“更多”菜单图标 (*...*)

3. 选择“下载原始视频”****。
