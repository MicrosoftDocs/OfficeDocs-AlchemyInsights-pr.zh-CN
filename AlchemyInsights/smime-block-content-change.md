# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook 现在将默认阻止 S/MIME 中的外部内容

我们在 Outlook for Windows 中引入了添加的安全措施, 通过在默认情况下阻止在 S/MIME 加密和签名的邮件中阻止外部内容, 从而防止在每个邮件的基础上进行覆盖。

有关完整的详细信息, 请参阅[Outlook For Windows 阻止 S/MIME 邮件中的外部内容](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e)。
