# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>使用 SaRA 诊断和解决 Outlook 身份验证问题

我们强烈建议您使用的[Outlook 总是询问我密码](https://aka.ms/SaRA-OutlookPwdPrompt)诊断受影响的计算机上来解决问题其中 Outlook 不断提示输入密码。诊断此[SaRA](https://diagnostics.office.com/#/)不自动的检查，并返回为您要用于解决检测到的任何问题可能的解决方案。
