# <a name="how-to-attach-files"></a>如何附加文件

若要附加经典 Outlook.com 中的文件：
  
1. 撰写新邮件或答复邮件。
    
2. 在撰写窗格的顶部，选择**附加**。 
    
若要附加 Outlook.com beta 中的文件：
  
1. 撰写新邮件或答复邮件。
    
2. 在撰写窗格的底部，选择 ![附加](media/da223d01-5fe6-448c-a3a3-e2b5262da4b9.png).
    
了解有关[如何附加 Outlook.com 中的文件](https://go.microsoft.com/fwlink/p/?linkid=2001702&amp;clcid=0x409)。
