# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>使用 OneDrive 同步功能同步 SharePoint 库

若要将 SharePoint 文件同步到本地硬盘，请完成以下步骤：

1. 导航到想要同步到本地硬盘的文档库。
2. 从菜单选项中选择“同步”按钮。
3. 按照分步说明操作以完成同步。
