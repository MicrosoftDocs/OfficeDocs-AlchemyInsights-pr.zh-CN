# <a name="create-a-new-folder"></a>创建一个新文件夹

1. 在导航窗格中的**文件夹**中，选择**新建文件夹**。 
    
2. 键入文件夹的名称，然后按 Enter。
    
若要了解详细信息，请参阅[Create Outlook.com 中的文件夹](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)。
