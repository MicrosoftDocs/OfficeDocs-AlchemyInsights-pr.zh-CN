# <a name="create-users-in-hybrid-deployments"></a>在混合部署中创建用户

如果您的组织是本地环境与 Office 365 之间的混合部署, 则需要在本地 Active Directory 中创建所有用户对象。 您可以在相应的组织单位 (OU) 中创建用户对象, 使其同步到您的 Office 365 组织中。
