# <a name="map-a-sharepoint-library-to-a-network-drive"></a>将 SharePoint 库映射到网络驱动器

若要在文件资源管理器中处理网站库, 我们建议使用 OneDrive 同步客户端, 而不是将库映射到网络驱动器。 驱动器映射只是一个临时解决方案。 使用同步客户端将使文件对用户可用, 即使它们处于脱机状态也是如此。 同步客户端还提供更好的性能, 允许对文件进行访问, 而无需使用["按需" 下载文件](https://support.office.com/article/Learn-about-OneDrive-Files-On-Demand-0E6860D3-D9F3-4971-B321-7092438FB38E), 并且不需要使用 Internet Explorer 进行安装。 
  
- [在 Windows 中使用新版 OneDrive 同步客户端来同步文件](https://go.microsoft.com/fwlink/?linkid=866427)。
    
- [Ways to work with site library files in File Explorer](https://go.microsoft.com/fwlink/?linkid=866291)（在文件资源管理器中处理网站库文件的方法）
    
- [配置映射的网络驱动器并对其进行疑难解答](https://support.microsoft.com/kb/2616712)
