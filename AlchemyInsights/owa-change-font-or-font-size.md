# <a name="change-font-or-font-size"></a>更改字体或字号

阅读电子邮件时, 可以使用浏览器的缩放功能放大或缩小屏幕上的文本。
  
撰写邮件时更改字体或字号:
  
1. 在撰写窗格的底部, 选择 "**字体**" 或 "**字号**" 图标。
    
2. 选择要使用的字体或字号。
    
如果在撰写窗格底部看不到 "格式选项" 菜单, 请选择 "**格式设置选项**" 图标。
  
有关详细信息, 请参阅[在 web 上的 Outlook 中更改字体大小](https://support.office.com/article/43a2137f-8c3c-46df-af4a-73a12c9bb86e)。
