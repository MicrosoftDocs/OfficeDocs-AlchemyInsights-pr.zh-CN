# <a name="import-contacts"></a>导入联系人

导入 Outlook.com 联系人之前, 您需要将其导出为.csv 文件其他电子邮件提供商。然后，按照以下说明。
  
1. 然后在页面的底部，选择**人员**。 
    
2. 在菜单栏上，选择**管理** \> **导入联系人**。 
    
3. 选择**Gmail** （即使您.csv 文件可能不会从 Gmail） 由于 Gmail 使用.csv 文件类型。 
    
4. 选择**浏览**，导航到保存导出的联系人的文件夹并选择**打开**。 
    
若要了解详细信息，请参阅[向 Outlook.com 帐户导入联系人](https://go.microsoft.com/fwlink/p/?linkid=873136)。
