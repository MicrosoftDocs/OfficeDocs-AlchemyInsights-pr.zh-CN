# <a name="set-passwords-to-never-expire"></a>设置密码永不过期

若要设置所有密码永不过期，转到[安全&amp;隐私](https://portal.office.com/adminportal/home#/settings/security)。单击**编辑**旁边**密码策略**，并打开到**上**滑块。
  
有关详细信息，请参阅：[设置为您的组织的密码到期策略。](https://support.office.com/article/0f54736f-eb22-414c-8273-498a0918678f)
  
[设置为永不过期的特定用户的密码](https://support.office.com/article/f493e3af-e1d8-4668-9211-230c245a0466)
