# <a name="whats-a-hub-site"></a>集线器站点是什么？

集线器站点可帮助您组织的站点，并为其提供一个常见导航和品牌。全局或 SharePoint 管理员将能够创建中心站点并指定哪些用户可以将其网站与中心站点关联。有关详细信息，请参阅[Office 365 中的新增功能 SharePoint 中心站点](https://go.microsoft.com/fwlink/?linkid=869388)。
