# <a name="conversion-of-microsoft-365-group-and-shared-mailboxes"></a>Microsoft 365 组和共享邮箱的转换

目前，没有办法在 Microsoft 365 组邮箱和共享邮箱之间进行转换。

请 [在此处](https://aka.ms/M365GroupToShared)帮助投票此功能。
