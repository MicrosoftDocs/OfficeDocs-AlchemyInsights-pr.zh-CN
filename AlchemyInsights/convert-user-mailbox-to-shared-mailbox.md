# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>将用户邮箱转换为会议室邮箱或设备邮箱

您还可以将邮箱从一种邮箱类型转换为另一种, 但必须使用 Exchange Online PowerShell 执行这些任务。 有关详细信息, 请参阅[转换邮箱](https://go.microsoft.com/fwlink/p/?LinkId=832875)。
