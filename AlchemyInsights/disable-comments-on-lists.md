# <a name="disable-comments-on-lists"></a>禁用列表上的注释

管理员可以通过更改 **Set-SPOTenant** PowerShell cmdlet 中的 **CommentsOnListItemsDisabled** 参数，在组织级别禁用该功能。

当前无法禁用站点或列表级别的注释。 我们希望在稍后的更新（可能在 2021 年第一季度）中提供这些控件。
