# <a name="import-contacts"></a>导入联系人

在将联系人导入到 Outlook.com 之前, 您需要将其从其他电子邮件提供商导出为 .csv 文件。 然后, 按照下面的说明进行操作。
  
1. 在页面底部, 选择 "**人员**"。 
    
2. 在菜单栏上, 选择 "**管理** \> **导入联系人**"。 
    
3. 选择**gmail** (即使 .csv 文件可能不来自 Gmail), 因为 Gmail 使用 .csv 文件类型。 
    
4. 选择 "**浏览**", 导航到保存导出的联系人的文件夹, 然后选择 "**打开**"。 
    
若要了解详细信息, 请参阅[将联系人导入到 Outlook.com 帐户](https://go.microsoft.com/fwlink/p/?linkid=873136)。
