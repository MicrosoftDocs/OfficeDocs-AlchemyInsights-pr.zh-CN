# <a name="deploy-office-apps"></a>部署 Office 应用程序

若要通过 Office 部署工具、Microsoft 终结点配置管理器将 Office 应用程序部署到多个最终用户，作为操作系统磁盘映像的一部分，或在使用 Windows Server 的远程桌面服务（RDS）或终端服务的共享计算机方案中，请考虑使用**Office 365 专业增强版部署顾问**中的逐步指南。
  
若要开始，请确保你已以租户管理员或全局管理员身份登录，然后单击此处： [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance)。
