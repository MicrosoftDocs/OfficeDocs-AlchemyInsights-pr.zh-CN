# <a name="deploy-office-apps"></a>部署 Office 的应用程序

部署到多个最终用户通过 Office 部署工具中，System Center Configuration Manager，Office 应用程序的一部分的操作系统磁盘映像，或共享的计算机方案中使用 Windows Server 远程桌面服务 (RDS) 或终端服务，请考虑在**Office 365 ProPlus 的部署顾问**中使用的循序渐进指南。
  
若要开始，请确保您已经为租户管理员或全局管理员身份登录，再单击此处： [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance)。
