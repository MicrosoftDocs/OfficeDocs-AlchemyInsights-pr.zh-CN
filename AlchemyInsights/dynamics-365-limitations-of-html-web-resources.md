# <a name="webpage-html-web-resources-limitations"></a>网页（HTML） web 资源限制

* HTML web 资源不能包含任何必须在服务器上执行的代码。 ASP.NET 页面不能作为 HTML web 资源进行上载。

* HTML web 资源只能接受有限数量的查询字符串参数。 详细信息：[将参数传递给 HTML web 资源](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
