# <a name="teams-outlook-add-in"></a>工作组 Outlook 加载项

## <a name="to-troubleshoot-a-missing-teams-outlook-add-in"></a>解决缺少的团队 Outlook 外接

1. 在管理门户中运行服务诊断。 
2. 如果服务诊断未指明任何问题，请下载并运行[Microsoft 支持和恢复助手](https://aka.ms/SaRA-TeamsAddInScenario)  >  **团队 Outlook 外接程序方案**，以诊断和更正受影响的用户计算机上的任何客户端问题。

若要手动解决缺少的团队 Outlook 外接程序，请查看可能导致团队外接程序不显示在列表中的潜在因素的列表。 有关详细信息，请参阅 [Outlook For Windows 中的团队会议外接程序不显示](https://docs.microsoft.com/microsoftteams/teams-add-in-for-outlook#teams-meeting-add-in-in-outlook-for-windows-does-not-show)。

## <a name="to-troubleshoot-scheduling-a-teams-meeting-on-behalf-of-someone-else-delegate"></a>若要解决代表他人安排的团队会议 (委派) 

若要安排团队会议代表其他人，请参阅 [Microsoft 团队和 Exchange Server 交互问题的疑难解答](https://docs.microsoft.com/microsoftteams/troubleshoot/known-issues/teams-exchange-interaction-issue)。
