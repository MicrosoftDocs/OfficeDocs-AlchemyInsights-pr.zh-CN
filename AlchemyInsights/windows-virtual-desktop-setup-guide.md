# <a name="set-up-windows-virtual-desktop"></a>设置 Windows 虚拟桌面

Windows 虚拟桌面是一项全面的桌面应用虚拟化服务，在云中运行。 它是唯一一个虚拟桌面基础结构 (VDI) ，可提供简化的管理、多会话 Windows 10、Microsoft 365 应用的优化，并支持远程桌面服务 (RDS) 环境。 在数分钟内将 Windows 桌面和应用部署并缩放到 Azure，并获取内置的安全性和合规性功能。

Windows [虚拟桌面设置指南为](https://go.microsoft.com/fwlink/?linkid=2146236) 管理员提供了规划资源和部署先决条件、设置指南和其他资源。
