# <a name="enable-multi-factor-authentication"></a>启用多重身份验证

1. 导航到**用户** \> **活动用户**
    
2. 单击**其他** \> **安装 Azure 多因素身份验证**。 
    
3. 找到您想要启用 MFA 用户。若要查看所有用户，您可能需要更改顶部的多因素身份验证状态视图。
    
4. 检查您想要启用的用户旁边的复选框。
    
5.  在适当的用户信息窗格中，在**快速步骤**下将请参阅 Enable 并管理用户设置。选择**启用**。 
    
6. 在对话框中，将打开，**单击启用多重身份验证**。
