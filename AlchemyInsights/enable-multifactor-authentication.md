# <a name="enable-multi-factor-authentication"></a>启用多重身份验证

1. 导航到 "**用户** \> **活动用户**"
    
2. 单击 "**更多** \> **安装 Azure 多因素身份验证**"。 
    
3. 查找要为其启用 MFA 的一个或一用户。 若要查看所有用户, 您可能需要更改顶部的 "多重身份验证状态" 视图。
    
4. 选中要启用的用户旁边的复选框。
    
5.  在 "右侧用户信息" 窗格中的 "**快速步骤**" 下, 您将看到 "启用和管理用户设置"。 选择" **启用**"。 
    
6. 在打开的对话框中,**单击 "启用多重身份验证**"。
