# <a name="delete-a-folder"></a>删除文件夹

1. 在左窗格中, 右键单击要删除的文件夹或子文件夹, 然后选择 "**删除**"。 
    
2. 选择 **"确定"** 以确认。 
    
若要了解详细信息, 请参阅[在 Outlook.com 中删除文件夹](https://go.microsoft.com/fwlink/p/?linkid=873134)。
