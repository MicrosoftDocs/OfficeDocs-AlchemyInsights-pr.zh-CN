# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>解决 Microsoft Intune 中 Apple MDM 推送证书的问题

查看下面列出的资源，以现在解决问题。
  
1. 查看 [本文档以](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get) 配置或续订 Apple MDM 推送证书。

2. 使用 [Intune 故障排除门户](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) 诊断和解决常见的注册失败。 有关详细信息 [，](https://docs.microsoft.com/intune/help-desk-operators) 请查看本文档。

3. 有关阻止注册的一系列常见错误及其解决方案，请查看以下文档：[故障排除指南](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune)和[疑难解答文档](https://docs.microsoft.com/troubleshoot/mem/intune/troubleshoot-device-enrollment-in-intune)。
