# <a name="troubleshoot-issues-with-apple-mdm-push-certificate-in-microsoft-intune"></a>解决问题中 Microsoft Intune 的 Apple MDM 推送证书

查看下列来解决此问题现在资源。 
  
1. 查看[此文档](https://docs.microsoft.com/intune/apple-mdm-push-certificate-get)配置或续订 Apple MDM 推送证书。 
    
2. 使用[Intune 疑难解答门户](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)诊断和解决常见注册失败。查看[此文档](https://docs.microsoft.com/intune/help-desk-operators)的详细信息。 
    
3. 查看这些文档中的每个阻止注册和解析的常见错误列表：[疑难解答指南](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune)和[疑难解答文档](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)。
