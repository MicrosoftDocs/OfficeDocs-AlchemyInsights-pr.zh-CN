# <a name="how-azure-active-directory-identity-protection-is-provisioned-and-deployed"></a>如何设置和部署 Azure Active Directory Identity Protection？

默认情况下，Azure Active Directory Identity Protection 的功能在租户级别为租户内的所有用户启用。

若要了解更多信息，请参阅 [什么是 Azure Active Directory Identity Protection？](https://go.microsoft.com/fwlink/?linkid=2130395)
