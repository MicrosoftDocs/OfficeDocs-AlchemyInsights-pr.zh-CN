# <a name="service-incident-in-progress"></a>正在进行的服务事件

为方便起见, 你可以在下面找到此事件的摘要:
  
> **事件编号:**[{IncidentId}](https://admin.microsoft.com/adminportal/home#/servicehealth)
 
> 影响说明: {ImpactDescription}
