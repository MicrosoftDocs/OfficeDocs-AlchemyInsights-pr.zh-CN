# <a name="service-incident-in-progress"></a>正在运行的服务事件

您可能会影响服务事件或顾问的正在调查。您可以通过转到[服务运行状况](https://admin.microsoft.com/adminportal/home#/servicehealth)页上检查的最新状态。 
  
为方便起见，您可以找到该事件下面的摘要：
  
> **事件编号：**[{} IncidentId](https://admin.microsoft.com/adminportal/home#/servicehealth)
    
> 影响说明: {ImpactDescription}
