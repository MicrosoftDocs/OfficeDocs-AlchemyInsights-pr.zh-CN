# <a name="what-is-autosave"></a>什么是自动保存？

"自动保存" 是 Excel、Word 和 PowerPoint for Office 365 订阅者提供的一项新功能, 可在工作时每隔几秒自动保存文件。 

有关常见问题的信息, 请参阅[什么是自动保存？](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
