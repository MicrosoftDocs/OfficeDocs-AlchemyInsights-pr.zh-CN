# <a name="join-a-team-by-code"></a>按代码加入团队

Microsoft 意识到了一个问题，即用户无法创建 **加入团队** 代码，或者使用 **通过代码** 功能加入团队。 作为解决方法，团队所有者可根据需要手动添加用户。
