# <a name="learn-about-site-compatibilityaffecting-changes-coming-to-microsoft-edge"></a>了解站点兼容性-影响即将对 Microsoft Edge 所做的更改

Web 在不断演变，以改进用户体验、安全性和隐私。 一些更改足以影响现有页面的功能。 Microsoft Edge 团队会随着思考的发展、日程表的强化和 [更改的公布来更新页面](https://go.microsoft.com/fwlink/?linkid=2135534)。
