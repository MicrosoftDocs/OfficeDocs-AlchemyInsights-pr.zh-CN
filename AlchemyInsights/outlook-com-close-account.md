# <a name="close-your-outlookcom-account"></a>关闭您的 Outlook.com 帐户

1. 转到 "[关闭帐户](https://go.microsoft.com/fwlink/p/?linkid=845493)" 页。
2. 按照页面上的说明操作，以确保您的帐户已准备好关闭，然后选择 "**下一步**"。
3. 阅读列表并选中复选框以确认您已阅读每个项目。
4. 在 "**选择原因**" 下拉列表中，选择要关闭帐户的原因。
5. 选择 "**标记帐户以**将其关闭"。

若要了解详细信息，请参阅[关闭你的 Outlook.com 帐户](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038?wt.mc_id=Office_Outlook_com_Alchemy)。
