# <a name="check-for-app-using-camera"></a>使用摄像头检查应用

1. 选择 " **开始**"，然后选择 " **设置" > 隐私 > 相机** (ms-设置：隐私-网络摄像头) 
2. 在 "应用程序" 列表中，找到 "当前正在使用" 的应用程序，然后关闭该应用程序。
