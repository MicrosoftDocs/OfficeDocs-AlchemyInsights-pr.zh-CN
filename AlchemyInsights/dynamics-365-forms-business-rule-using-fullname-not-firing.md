# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>统一 inteface 应用程序不支持完整名称字段

基于统一界面的应用程序中不支持使用 **全名** (fullname) 字段的操作或条件。 或者，也可以使用 **名** 为 "名字" ("姓氏) " 和 " **姓** " (lastname) "字段的操作或条件。
