# <a name="changes-to-external-user-access"></a>对外部用户访问的更改

启动年 3 月 23 2018年外部用户将不再能够访问即使他们已邀请或添加到授权组已与每个人、 所有经过身份验证用户或所有表单用户共享的内容。外部用户仅能访问已与用户或用户所属的组共享的内容。如果您想要保留现有的设置，请参阅：https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
