# <a name="report-suspicious-urls"></a>报告可疑 URL

若要提交可疑 URL，请按照以下步骤操作：

1. 在" **新建提交** "飞出中，输入完整 URL，包括协议 (https：//) 。
2. 如果选择" **应该已被阻止"，** 请指定 URL 是 **网络钓鱼** 还是 **恶意软件**。
3. 选择“**提交**”。
