# <a name="report-suspicious-attachments"></a>报告可疑附件

1. 在" **新建提交** "飞出中，选择 **"** 附件"，然后上载要扫描的文件。
    > [!NOTE]
    > 目前存在 3 MB 的限制。 如果文件超过 3 MB，请考虑改为使用电子邮件 ID 提交它。
2. 如果选择 **"应该已被阻止"，** 它将自动将项目标记为 **"恶意软件"。**
3. 选择“**提交**”。
