# <a name="i-dont-see-automatic-replies"></a>我看不到自动答复

如果在选择"文件"后看不到"自动答复"，你可能会使用 Gmail、Yahoo 或其他不支持 Outlook 自动答复功能 POP 或 IMAP 帐户。  您可以设置将答复传入邮件的规则，但仅在 Outlook 保持运行状态时。
