# <a name="update-dns-settings"></a>更新 DNS 设置

更新 DNS 设置涉及将 Office 365 为你提供的值复制并粘贴到域注册机构或 DNS 托管提供商中。 为此，请执行下列步骤：

1. 将"更新 DNS 设置"页上显示的值复制 **并** 粘贴到 DNS 托管提供商中。
2. 单击 **"验证"。**
3. 单击“完成”。
