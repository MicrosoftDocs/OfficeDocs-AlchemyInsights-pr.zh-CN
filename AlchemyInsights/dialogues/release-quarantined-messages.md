# <a name="release-quarantined-messages"></a>释放隔离邮件

可以将隔离的邮件 (一组) 收件人，也可以仅将邮件释放给从所有收件人列表中选择的特定人员。

1. 转到安全与 [&合规中心](https://go.microsoft.com/fwlink/p/?linkid=2077143)。
2. 转到 **"威胁管理**  >  **审阅**  >  **隔离"。**
3. 选择要 () 的邮件。
4. 释放邮件后，选择 **"刷新** "以刷新数据，然后双击该消息。 应看到邮件已释放给预期收件人。
