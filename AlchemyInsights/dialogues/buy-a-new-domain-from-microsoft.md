# <a name="buy-a-new-domain-from-microsoft"></a>从 Microsoft 购买新域

若要从 Microsoft 购买新域，请执行以下步骤：

1. 导航到 **">域**"，然后单击 **"购买域"。** 
2. 按照向导中的指示进行操作。
