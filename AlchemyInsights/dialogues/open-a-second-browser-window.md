# <a name="open-a-second-browser-window"></a>打开第二个浏览器窗口

若要打开第二个浏览器窗口，请执行以下步骤：

1. 在阅读此 (的第一个浏览器窗口中) ，确保域向导已打开。 如果未单击上述按钮以启动域向导，则可以通过进入"域"和"添加域"> **域向导**。

2. 打开第二个浏览器窗口。 第二个浏览器中，将按照后续步骤操作。
