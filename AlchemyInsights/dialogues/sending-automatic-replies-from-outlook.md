# <a name="sending-automatic-replies-from-outlook"></a>从 Outlook 发送自动答复

- 有两种方法可发送自动外出答复，具体取决于你拥有的电子邮件帐户类型。
- 单击 **"****文件"，** 然后根据您的 Outlook 版本选择"自动答复"。
