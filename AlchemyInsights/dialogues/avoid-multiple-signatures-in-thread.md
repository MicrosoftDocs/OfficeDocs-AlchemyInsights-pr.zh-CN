# <a name="avoid-multiple-signatures-from-being-added-in-an-email-conversation"></a>避免在电子邮件对话中添加多个签名

若要获取每个电子邮件线程仅添加一次免责声明或签名，您需要更改 Exchange 管理中心中的设置。

1. 转到 [Exchange 管理中心。](https://go.microsoft.com/fwlink/p/?linkid=2059104)
2. 选择 **"更多** 选项"可添加在免责声明中查找唯一文本的例外。 这可确保仅将免责声明添加到原始邮件中。
