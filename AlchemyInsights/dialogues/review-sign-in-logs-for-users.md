# <a name="review-sign-in-logs-for-users"></a>查看用户的登录日志

下面是如何进行此操作：

1. 转到 **管理中心**  >  **[Azure Active Directory](https://go.microsoft.com/fwlink/p/?linkid=2067268)**。
1. 选择 **Azure Active Directory**  >  **登录。**
1. 如果你还没有 Azure AD P1 或 P2 订阅，系统将提示你启动免费高级试用版。 如果需要，请单击"开始 **免费高级版试用版**"，然后在 Azure AD Premium P2 框中，选择 **"免费试用版**  >  **激活"。** 免费试用期为 30 天。
