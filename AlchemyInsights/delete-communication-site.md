# <a name="delete-a-communication-site"></a>删除通信网站

1. 转到网站。
    
2. 单击右上角的 "设置" 图标。
    
3. 单击 "**网站设置**"。 在 "**网站信息**" 下, 单击 "**删除此网站**", 然后单击 "**删除**"。
