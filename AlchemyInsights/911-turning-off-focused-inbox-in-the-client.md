# <a name="turn-off-focused-inbox-in-email-clients"></a>在电子邮件客户端关闭中心收件箱

若要关闭中心收件箱中 Outlook 2016，在 web 上的 Outlook，Outlook.com/Hotmail.com、 Outlook 2016 Mac，或邮件的 Windows 10，请参阅[关闭中心收件箱](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx)。
  
若要关闭中心收件箱的组织中所有人，请参阅[配置中心收件箱的组织中的每个人](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx)。
