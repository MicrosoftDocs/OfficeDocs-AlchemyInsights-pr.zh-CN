# <a name="policy-error-when-sharing-a-calendar"></a>共享日历时出现策略错误

如果您尝试共享日历，并且收到错误 "策略不允许将此级别的权限授予一个或多个收件人 (s) "，请参阅 [本主题](https://support.microsoft.com/help/3187524/policy-does-not-allow-granting-permissions-at-this-level-to-one-or-mor)。
