# <a name="import-contacts"></a>导入联系人

在将联系人导入到 web 上的 Outlook 之前, 你需要将其从其他电子邮件提供商导出为 .csv 文件。 然后, 按照下面的说明进行操作。

1. 在页面底部, 选择 "人员" 图标以转到 "人员" 页面。
2. 在工具栏上, 选择 "**管理** > **导入联系人**"。
3. 选择 "**浏览**", 选择您的 CSV 文件, 然后选择 "**打开**"。
4. 选择“**导入**”。

若要了解详细信息, 请参阅[将联系人导入到 Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web)。
