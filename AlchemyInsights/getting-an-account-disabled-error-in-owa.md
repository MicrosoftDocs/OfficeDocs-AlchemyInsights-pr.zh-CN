# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>在 web 上的 Outlook 中获取帐户禁用错误？

如果遇到错误, 则**** 当您尝试打开 web 上`X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException`的 Outlook (以前称为 outlook web App) 时, 您的帐户已被禁用。您的组织管理员可能已禁用您对 web 上的 outlook 的访问。 您的管理员可以按照[启用或禁用适用于邮箱的 Outlook Web App](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)主题中的步骤来修复此错误。
