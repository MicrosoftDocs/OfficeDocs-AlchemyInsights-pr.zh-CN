# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>在 web 上的 Outlook 中获取帐户禁用错误？

如果遇到错误，则**Your account has been disabled** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` 当您尝试在 (Web 上打开 outlook 时，您的帐户已被禁用（以前称为 outlook web App) ）时，您的组织管理员可能已经禁用了您对 Web 上的 outlook 的访问。 您的管理员可以按照 [启用或禁用适用于邮箱的 Outlook Web App](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)主题中的步骤来修复此错误。
