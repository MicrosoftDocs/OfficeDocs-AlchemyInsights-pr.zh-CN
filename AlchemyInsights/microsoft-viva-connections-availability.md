# <a name="microsoft-viva-connections-availability"></a>Microsoft Viva 连接可用性

Microsoft Viva Connections 将在 2021 年上半年提供桌面，在 2021 年夏提供移动版。 其他功能将在 2022 年推出。

有关详细信息，请参阅 Microsoft [Viva Connections 如何帮助所有人保持参与和了解情况](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-connections-helps-everyone-to-stay-engaged-and/ba-p/2107009)。
