# <a name="change-a-domain-name-letter-case-or-spelling"></a><span data-ttu-id="3e2f2-102">更改域名的字母大小写或拼写</span><span class="sxs-lookup"><span data-stu-id="3e2f2-102">Change a domain name letter case or spelling</span></span>

<span data-ttu-id="3e2f2-103">若要更改自定义域名的拼写，例如更改为全部使用小写字母或大小写字母混合，必须删除该域，然后使用所需的字母大小写或拼写形式重新添加。</span><span class="sxs-lookup"><span data-stu-id="3e2f2-103">To change the spelling of a custom domain name, for example to all lowercase letters or a mix of lower and upper case letters, the domain must be removed and then re-added with the desired letter case or spelling.</span></span>

1. <span data-ttu-id="3e2f2-104">在“[域](https://admin.microsoft.com/Adminportal#/Domains)”页面上，选择相应的域，然后选择“**删除域**”。</span><span class="sxs-lookup"><span data-stu-id="3e2f2-104">On the [Domains page](https://admin.microsoft.com/Adminportal#/Domains), choose the domain, and then select  **Remove domain**.</span></span></br>

    <span data-ttu-id="3e2f2-105">有关详细说明，请参阅[如何删除域](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide)。</span><span class="sxs-lookup"><span data-stu-id="3e2f2-105">For for more detailed instructions, see [How to remove a domain](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain?view=o365-worldwide).</span></span>

2. <span data-ttu-id="3e2f2-106">使用“[添加域向导](https://admin.microsoft.com/Adminportal#/Domains/Wizard)”重新添加具有正确拼写和字母大小写的域。</span><span class="sxs-lookup"><span data-stu-id="3e2f2-106">Re-add the domain with the correct spelling and letter case by using the [Add Domain Wizard](https://admin.microsoft.com/Adminportal#/Domains/Wizard).</span></span>

<span data-ttu-id="3e2f2-107">注册后，既无法更改字母大小写，也无法重命名 onmicrosoft.com 域或通过 Microsoft 购买的域。</span><span class="sxs-lookup"><span data-stu-id="3e2f2-107">You can't change letter case or rename the onmicrosoft.com domain, or a domain purchased through Microsoft, after sign-up.</span></span>
