# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive 在处理更改时被卡住

如果 OneDrive 显示 "处理更改" 的时间很长，则可能是因为打开了联机文件、大量文件在队列中，或者是大型文件同步。 阅读 OneDrive 因建议的步骤和详细信息而  [停滞在 "处理更改" 上](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) 。

如果您需要更多帮助，您可以  [通过电子邮件发送 OneDrive 支持](https://go.microsoft.com/fwlink/p/?LinkId=528676)。
