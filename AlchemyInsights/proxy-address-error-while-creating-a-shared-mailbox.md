# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>创建共享的邮箱时的代理地址错误

如果您尝试创建共享的邮箱，并收到此错误"的"SMTP:alias@domain.com"已被...使用代理地址"，选择您的电子邮件地址已被占用通过您的组织中的另一个邮箱。
  
您需要查找用户、 组或共享的邮箱已此电子邮件地址并将其删除或更改其电子邮件地址。然后您可以使用的释放电子邮件地址来创建共享的邮箱。在主页上使用搜索来查找它。
  
如果您不希望删除现有的电子邮件地址，选择您共享邮箱的新电子邮件地址。
