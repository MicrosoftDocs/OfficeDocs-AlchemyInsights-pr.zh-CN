# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>创建共享邮箱时出现代理地址错误

如果您尝试创建共享邮箱并收到此错误 "代理地址" SMTP:alias@domain.com "已在使用中 ...", 则您选择的电子邮件地址已由组织中的另一个邮箱使用。
  
您需要查找具有此电子邮件地址的用户、组或共享邮箱, 并将其删除或更改其电子邮件地址。 然后, 您可以使用已释放的电子邮件地址创建共享邮箱。 使用主页上的 "搜索" 来查找该页面。
  
如果您不想删除现有的电子邮件地址, 请为您的共享邮箱选择一个新的电子邮件地址。
