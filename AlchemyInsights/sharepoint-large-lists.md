# <a name="work-with-large-lists-and-libraries-in-sharepoint"></a>处理大型列表和库 SharePoint 中

SharePoint 列表和库可以包含最多 30 万个项目，但当拥有 5000 个以上的项目，当您尝试使用它们时，您可能会看到一个列表视图阈值的错误。此阈值是为了保持服务的性能。不能更改。若要避免达到此阈值，您可以[添加索引](https://go.microsoft.com/fwlink/?linkid=867784)，[用户筛选器来更改视图](https://go.microsoft.com/fwlink/?linkid=867786)，将项目组织到文件夹，或使用同步的文档库本地计算机上。 
  
[在列表视图阈值的工作的更多提示](https://go.microsoft.com/fwlink/?linkid=867787)
