# <a name="need-to-create-an-email-signature"></a>是否需要创建电子邮件签名？

如果您拥有 Outlook 帐户或 web （工作或学校）帐户，则可以创建签名并将其自动添加到所有传出邮件，或将其选择性地添加到特定的传出邮件中。 请注意，您的电子邮件签名在 Outlook 和 web 上的 Outlook 中有所不同。 若要创建电子邮件签名，请参阅[在 outlook 中创建邮件签名并向其添加签名](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)，或[在 Web 上的 outlook 中创建并添加电子邮件签名](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)。
