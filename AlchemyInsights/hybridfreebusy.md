# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>混合环境中的日历忙/闲问题

如果用户在混合环境中查看日历的忙/闲信息时遇到问题，请重新运行 "混合配置" 向导。 有关 "混合配置" 向导的详细信息，请参阅[混合配置向导](https://go.microsoft.com/fwlink/p/?linkid=528149)。
