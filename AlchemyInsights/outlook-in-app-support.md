# <a name="in-app-support-in-outlook-for-windows"></a>Outlook for Windows 中的应用程序内支持

作为租户管理员，你可以在应用程序中接收适用于 Outlook for Windows 中的问题的应用程序支持。 在 Outlook 中，选择 "**帮助**" "  >  **联系支持**"。 输入你的问题，Outlook for Windows 支持团队的成员将回复你的消息。
