# <a name="rename-a-sharepoint-site"></a>重命名 SharePoint 网站

按照[本文](https://docs.microsoft.com/sharepoint/change-site-address)中的步骤将网站的名称（例如，从https://contoso.sharepoint.com/sites/sample1到https://contoso.sharepoint.com/sites/sample2）更改为。

**注意：** 您不能在 SharePoint Online URL 中更改域名（在上面的示例中，URL 的 contoso 部分）。 有关详细信息，请参阅[更改你的域](https://go.microsoft.com/fwlink/?Linkid=2018696)。
