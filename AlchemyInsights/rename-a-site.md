# <a name="rename-a-site"></a>重命名的网站

若要从更改站点名，例如https://contoso.sharepoint.com/sites/sample1到https://contoso.sharepoint.com/sites/sample2，您需要创建新网站，然后将所有内容从旧网站都移到新数据库。有关移动文档库中的项目的信息，请参阅[从 SharePoint 文档库项目移动或复制](https://go.microsoft.com/fwlink/?Linkid=2018691)。
  
注意： 如果您试图更改在 SharePoint Online URL 中的域名 （即，上面的 URL 的 contoso 一部分），这不可能。请参阅[更改您的域](https://go.microsoft.com/fwlink/?Linkid=2018696)。
