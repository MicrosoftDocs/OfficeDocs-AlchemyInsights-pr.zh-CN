# <a name="rename-a-sharepoint-site"></a>重命名 SharePoint 网站

若要更改网站的名称，例如从https://contoso.sharepoint.com/sites/sample1更改为https://contoso.sharepoint.com/sites/sample2，您需要创建一个新网站，然后将旧网站的所有内容移到新网站。 有关在文档库中移动项目的信息，请参阅[从 SharePoint 文档库移动或复制项目](https://go.microsoft.com/fwlink/?Linkid=2018691)。
  
注意：如果您尝试在 SharePoint Online URL 中更改域名（即上述 URL 的 contoso 部分），则这是不可能的。 请参阅[更改你的域](https://go.microsoft.com/fwlink/?Linkid=2018696)。
