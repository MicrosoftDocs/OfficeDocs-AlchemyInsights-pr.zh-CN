# <a name="rename-a-site"></a>重命名的网站

若要更改网站的名称，例如，从 https://contoso.sharepoint.com/sites/sample2 更改为 [](https://go.microsoft.com/fwlink/?Linkid=2018691)，需要创建一个新网站，然后将所有内容从旧网站移动到新网站。有关在文档库中移动项目的信息，请参阅从 SharePoint 文档库移动或复制项目。
  
注意： 如果您试图更改在 SharePoint Online URL 中的域名 （即，上面的 URL 的 contoso 一部分），这不可能。请参阅[更改您的域](https://go.microsoft.com/fwlink/?Linkid=2018696)。
