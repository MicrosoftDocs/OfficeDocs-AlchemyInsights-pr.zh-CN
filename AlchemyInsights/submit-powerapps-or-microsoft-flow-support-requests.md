# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>提交 PowerApps 或 Microsoft 流支持请求

如果你是 **PowerApps** 或 **Microsoft Flow** 客户，请通过新的 [Power platform 管理中心](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819)提交你的票证。 新门户提供了针对 PowerApps 和其他 Power platform 产品而优化的协助支持的更直接途径。

[提交 Microsoft 流支持请求](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
