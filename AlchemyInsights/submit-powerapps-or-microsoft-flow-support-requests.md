# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>提交 PowerApps 或 Microsoft Flow 支持请求

如果你是 **PowerApps 或** **Microsoft Flow** 客户，请通过新的 [Power 平台管理中心提交票证](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819)。 新门户为针对 PowerApps 和其他 Power 平台产品优化的辅助支持提供了更直接的路径。

[提交 Microsoft Flow 支持请求](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
