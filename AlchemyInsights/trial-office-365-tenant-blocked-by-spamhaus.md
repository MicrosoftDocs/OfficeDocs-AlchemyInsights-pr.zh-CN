# <a name="error-when-a-microsoft-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Microsoft 365 试用版用户发送电子邮件时出现错误：已使用 Spamhaus 阻止客户端主机

您似乎正在从新的 Microsoft 365 试用组织发送电子邮件。 为了帮助防止滥用，我们阻止已知已被泄露的 IP 地址。 开始使用付费许可证后，不再强制执行此特定检查。 您可以通过从 internet 服务提供商 (ISP) 或从 [Spamhaus](https://go.microsoft.com/fwlink/p/?linkid=123245) 网站删除阻止来请求新的 IP 地址，从而获得暂时的安全阀。
