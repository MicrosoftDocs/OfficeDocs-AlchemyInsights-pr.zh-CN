# <a name="need-help-with-microsoft-365-business-premium-voice"></a>需要 Microsoft 365 商业高级版语音帮助？

Microsoft 365 商业高级语音使中小型组织可以轻松将 Microsoft Teams 转变为功能强大且灵活的电话系统。 有关商务语音详细信息，请参阅什么是 [Microsoft 365 商业高级语音？](https://docs.microsoft.com/microsoftteams/business-voice/whats-business-voice)

如果你已获取 Microsoft 365 商业高级版语音[](https://docs.microsoft.com/microsoftteams/business-voice/use-getting-started-wizard)，"入门"向导可以帮助你开始操作。
