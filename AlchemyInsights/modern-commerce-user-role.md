# <a name="modern-commerce-user-role"></a>新式商业用户角色

新式商业用户角色使某些用户有权访问 Microsoft 365 管理中心并查看“**主页**”、“**计费**”和“**支持**”的左侧导航条目。 这些区域中的可用内容由分配给用户的特定于商业的角色控制，用于管理他们为自己或你的组织购买的产品。 这可能包括支付帐单等任务，或用于访问计费帐户和计费对象信息。

具有新式商业用户角色的用户通常在其他 Microsoft 购买系统中具有管理权限，但没有全局管理员或计费管理员权限。

若要了解有关此角色和其他管理角色的详细信息，请参阅 [Azure Active Directory 中的管理员角色权限](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#modern-commerce-administrator)。
