# <a name="use-your-browser-to-check-spelling"></a>使用浏览器中检查拼写

没有在 Outlook.com 中的拼写检查选项。要检查拼写，您将需要使用 web 浏览器。拼写检查位于 Microsoft 边缘、 Internet Explorer 10 和更高版本，以及当前版本的 Firefox、 Chrome 和 Safari 中。检查您的 web 浏览器，若要了解有关如何进行拼写检查选项。
