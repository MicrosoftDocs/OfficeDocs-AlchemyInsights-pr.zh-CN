# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>在 Skype 的 Outlook.com 中与家人和朋友交谈

Outlook.com 客户可以使用 Skype 中的聊天、语音和视频功能与朋友和家人交谈。 当您[合并您的 Microsoft 和 Skype 帐户](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409)时，您将在 Outlook.com 中自动拥有对 Skype 网页的访问权限。
  
了解有关如何[在 Skype 的 Outlook.com 中与家人和好友交谈](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409)的详细信息。
