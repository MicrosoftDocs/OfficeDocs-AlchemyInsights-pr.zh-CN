# <a name="delay-receiving-updates-to-microsoft-365-apps-for-enterprise"></a>延迟接收 Microsoft 365 企业应用版更新

如果你的特定业务需求需要延迟从 Semi-Annual 频道接收功能更新，可以将应用配置为不接收这些功能更新，但仍会收到最新的安全更新。

若要了解更多信息，请参阅 Microsoft [365](https://go.microsoft.com/fwlink/?linkid=2109533)企业应用版Semi-Annual频道延迟接收功能更新。
