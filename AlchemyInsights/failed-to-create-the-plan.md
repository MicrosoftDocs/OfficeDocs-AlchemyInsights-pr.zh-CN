# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>尝试在 Microsoft 团队中创建 Planner 选项卡时出现 "无法创建计划" 错误

如果尝试在 Microsoft 团队中创建 Planner 选项卡时收到 "未能创建计划" 错误，请按照以下步骤操作，以确保管理员是该组的成员：

1. 在 "管理中心" 中，转到 "**组**"  >  [组](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups)。 
2. 选择该组，然后在“成员”**** 选项卡上，选择“查看全部并管理成员”****。
3. 添加成员，然后选择 " **保存**"。
