# <a name="banned-sending-ip"></a>禁止发送 IP

将邮件发送到 Microsoft 365 时，发件人可能会遇到此错误。 发件人需要访问https://sender.office.com并按照说明进行操作。  如果他们没有从门户收到验证电子邮件，请让他们检查垃圾邮件文件夹和/或尝试使用不同的电子邮件地址。
