# <a name="defender-tamper-protection"></a>Defender 防篡改保护 

使用防篡改保护，将阻止恶意应用程序采取如下操作：

- 禁用病毒和威胁防护
- 禁用实时保护
- 关闭行为监视
- 禁用防病毒 (例如 IOfficeAntivirus (IOAV) # A3
- 禁用云提供的保护
- 删除安全智能更新

有关更多详细信息，请参阅 [使用防篡改保护保护安全设置](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/prevent-changes-to-security-settings-with-tamper-protection)。
