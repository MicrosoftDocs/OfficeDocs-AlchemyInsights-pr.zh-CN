# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>在混合环境中的委派的邮箱权限

根据您在内部部署组织中安装的 Exchange 的版本，您可能需要采取一些额外的步骤，设置您的内部部署组织与 Office 365 中 Exchange Online 之间的委派的邮箱权限。有关如何设置委派权限的详细信息，了解一下[配置 Exchange 以支持混合部署中的委派的邮箱权限](https://technet.microsoft.com/en-us/library/mt784505%28v=exchg.150%29.aspx)。
