# <a name="sharepoint-files-are-missing"></a>SharePoint 文件丢失

如果大量 OneDrive 或 SharePoint 文件被恶意软件删除、覆盖、损坏或感染，我们会为用户提供各种选项来帮助恢复过程。

有关详细信息，请参阅如何在 [SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2110774)和 OneDrive for Business 中恢复缺失、已删除或损坏的项目。
