# <a name="create-a-folder"></a>创建文件夹

1. 在左窗格中, 选择 "**文件夹**" 旁边的 "**新建文件夹**"。 
    
2. 在 "新建文件夹" 框中, 输入文件夹的名称, 然后按 enter。
    
若要了解详细信息, 请参阅[在 Outlook.com 中创建文件夹](https://go.microsoft.com/fwlink/p/?linkid=873114)。
