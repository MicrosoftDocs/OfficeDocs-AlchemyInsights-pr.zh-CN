# <a name="restore-items-deleted-from-sharepoint"></a>还原从 SharePoint 中删除的项目

从 Sharepoint 中删除项目或网站时, 并不会&rsquo;立即将其删除。 已删除邮件在一段时间内进入回收站。 在这段时间内, 您可以将删除的项目还原到其原始位置。 有关详细信息, 请访问下面的链接。</p> <p><strong>从回收站还原已删除的文件、文件夹和网站。</strong></p> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b?ui=en-US&amp;rs=en-US&amp;ad=US">Restore items in the Recycle Bin of a SharePoint site</a> （还原 SharePoint 网站回收站中的项目）</li> </ul> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f">在 OneDrive 中还原已删除的文件或文件夹</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/sharepoint/restore-deleted-site-collection">还原已删除的网站集 (包括组、通信和其他网站)</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/en-us/onedrive/restore-deleted-onedrive">还原已删除的 OneDrive 网站</a></li> </ul> <p>对于批量回收站操作, 管理员可以考虑使用<a href="https://docs.microsoft.com/en-us/powershell/sharepoint/sharepoint-pnp/sharepoint-pnp-cmdlets?view=sharepoint-ps">Sharepoint ONLINE PNP。</a></p> <p><strong>文件还原功能</strong></p> <p>如果大量 OneDrive 或 Sharepoint 文件被恶意软件删除、覆盖、损坏或感染, 则可以使用 "文件还原" 功能将整个 OneDrive 或 Sharepoint 库还原到&nbsp;以前的某个时间。</p> <ul> <li><a href="https://support.office.com/en-us/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15">还原 OneDrive 库</a></li> <li><a href="https://support.office.com/en-us/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&amp;rs=en-US&amp;ad=US">还原文档库</a></li> </ul> <p><strong>注意:</strong>除了实际删除之外, SharePoint Online 还会将所有内容的备份保留14天。 如果无法通过 "回收站" 或 "文件还原" 来还原内容, 管理员可以联系 Microsoft 支持, 以在14天内的任何时间请求还原。&nbsp;仅可对网站集或子网站 (而不是特定文件、列表或库) 完成备份的还原。</p>
