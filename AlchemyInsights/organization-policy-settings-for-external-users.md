# <a name="organization-policy-settings-for-external-users"></a>外部用户的组织策略设置

当用户尝试与组织的外部用户共享时，他们可能会看到以下警告消息： 

   **你的组织的策略不允许你与这些用户共享。转到 Microsoft 365 管理中心中的 "外部共享" 以启用它。** 

这通常发生在尝试与外部用户共享或与新外部用户共享时。 有关详细信息，请参阅下面的文章：

[你的组织的策略不允许你与这些用户共享](https://docs.microsoft.com/sharepoint/support/administration/organization-policies-do-not-allow-you-to-share-with-users-error)
