# <a name="email-aliases-on-mobile-devices"></a>在移动设备上的电子邮件别名

电子邮件别名是另一个邮箱配置的电子邮件地址，因此邮件发送到用户的别名将被直接发送到其邮箱，可在其移动设备上。不需要进行任何其他配置。
  
但用户无法回复邮件或将使用别名 as 新邮件发送其发件人地址。所有传出邮件将从其主电子邮件地址。
