# <a name="new-feature-get-feedback-about-spam-judgments"></a>新功能: 获取有关垃圾邮件判断的反馈

作为管理员, 你现在可以在[提交资源管理器](https://protection.office.com/reportsubmission)中轻松地将可疑电子邮件、url 和内容报告给 Microsoft。 您甚至可以使用它来帮助了解为什么错误地标记了合法邮件。 有关详细信息, 请参阅此[通知](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622)。

若要使其 easer, 您可以允许您的用户通过[报告邮件外接程序](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)直接向 Microsoft 报告。
