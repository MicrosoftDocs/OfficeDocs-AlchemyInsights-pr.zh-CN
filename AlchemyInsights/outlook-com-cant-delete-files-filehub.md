# <a name="cant-delete-files-or-photos-from-files"></a>无法从文件中删除文件或照片

文件显示附加到邮箱中邮件的文件和照片。 您不能直接从文件中删除文件或照片。 若要删除文件和照片, 您需要删除其附加的电子邮件:

有关详细信息, 请参阅[如何在 Outlook.com 中删除文件或照片](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)。
