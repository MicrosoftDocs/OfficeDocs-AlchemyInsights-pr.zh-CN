# <a name="credit-check-status-request"></a>信用检查状态请求

你的订单正在等待信用验证。 这通常在24小时内完成, 但可能需要长达48小时。 完成后, 你的订阅状态将从 "挂起" 更改为 "活动", 并且服务将可供使用。
