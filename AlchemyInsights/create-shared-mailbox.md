# <a name="create-a-shared-mailbox"></a>创建共享邮箱

共享的邮箱允许多个用户发送和接收电子邮件别名，如 sales@contoso.com，而不分配另一个许可证。若要创建共享的邮箱，请转到[共享的电子邮件指南](https://portal.office.com/adminportal/home)。
  
请参阅文章[创建共享邮箱](https://support.office.com/en-us/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)的提示和有关使用共享的邮箱的详细信息。
