# <a name="create-a-shared-mailbox"></a>创建共享邮箱

共享邮箱允许多个人发送和接收别名电子邮件，如 sales@contoso.com，无需分配其他许可证。 

有关使用共享邮箱的提示和详细信息，请参阅文章[创建共享邮箱](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)。
