# <a name="create-a-shared-mailbox"></a>创建共享邮箱

共享邮箱允许多个人发送和接收别名电子邮件，如 sales@contoso.com，无需分配其他许可证。 若要创建共享邮箱, 请转到[共享电子邮件指南](https://portal.office.com/adminportal/home)。
  
有关使用共享邮箱的提示和详细信息, 请参阅文章[创建共享邮箱](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)。
