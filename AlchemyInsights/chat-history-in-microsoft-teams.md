# <a name="viewing-chat-history-in-microsoft-teams"></a>在 Microsoft Teams 中查看聊天历史记录

若要查看 Teams 聊天历史记录，[可以使用](https://sip.protection.office.com/insightdashboard)安全[](https://sip.protection.office.com/contentsearchbeta?ContentOnly=1)& 合规性中心的内容搜索，可在其中搜索与 Microsoft 365 组或 Microsoft 团队关联的邮箱。 

请参阅 [搜索 Microsoft Teams 和 Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/content-search) 组，了解安全与合规中心中的搜索功能。 

**注意：** 目前，无法直接从 Microsoft Teams 中备份、导出、打印和存档对话历史记录。 请在 [UserVoice](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20)中对此功能投票。
