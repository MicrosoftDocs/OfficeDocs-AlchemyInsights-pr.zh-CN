[将应用程序身份验证从 Active Directory 联合身份验证服务移动到 Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) - 开发人员侧重于将应用程序从 Active Directory 联合身份验证服务迁移到 Azure Active Directory 的指南。
