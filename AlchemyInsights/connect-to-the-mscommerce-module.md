# <a name="connect-to-the-mscommerce-module"></a>连接到 MSCommerce 模块

您必须先连接到 MSCommerce 模块，然后才能查看或设置 AllowSelfServicePurchase 策略。  

若要连接到 MSCommerce 模块，请在 PowerShell 提示符下 (PS C： \) ，输入以下命令：

`Connect-MSCommerce`

这将打开登录对话框。 输入您的用户名和密码以登录。

**注意：** &nbsp; &nbsp;用于登录的帐户必须是公司或帐单管理员。
