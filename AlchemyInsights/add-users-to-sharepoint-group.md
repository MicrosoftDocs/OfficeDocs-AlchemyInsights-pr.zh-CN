# <a name="add-users-to-a-sharepoint-group"></a>将用户添加到 SharePoint 组

SharePoint 组是对网站和内容具有相同权限集的用户的集合。 您可以使用组轻松地将相同的权限级别一次性分配给多个用户，而不是一次向一个人分配权限。

- [将用户添加到组](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [使用管理中心添加或删除 Office 365 组中的成员](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
