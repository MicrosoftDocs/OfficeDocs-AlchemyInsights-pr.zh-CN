# <a name="learn-about-progressive-web-apps-pwas-on-microsoft-edge-chromium"></a>了解 Microsoft Edge (Chromium) 上的渐进 web 应用 (PWA) 

PWA 是逐步增强的网站，其功能与支持它们的平台上的本机应用类似。 PWA 结合了 Web 和本机应用的最佳质量。

从 Web 平台的角度来看，Microsoft Edge 上的 [PWA (Chromium) ](https://go.microsoft.com/fwlink/?linkid=2135193) 基于标准。 它们允许用户直接从浏览器安装应用，并且不需要基于 Microsoft Store 的部署或注册。 桌面 PWA 在 Microsoft Edge (Chromium) 可用的任何平台上都受支持。
