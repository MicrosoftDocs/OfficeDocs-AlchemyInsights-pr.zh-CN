# <a name="your-users-mailbox-is-being-created"></a>正在创建你的用户的邮箱

在新邮箱准备就绪前, 可能需要一小时的时间。
  
如果用户的邮箱的 "正在准备" 的时间超过一小时, 则可以尝试删除并重新创建该用户, 或使用 "我们称之为你" 选项与支持工程师交谈。
