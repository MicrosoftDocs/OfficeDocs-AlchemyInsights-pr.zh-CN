# <a name="remove-an-extension-from-microsoft-edge"></a>从 Microsoft Edge 中删除扩展

若要从 Microsoft Edge 中删除扩展，请：

1. 打开 Microsoft Edge。
2. Select **... > Extensions**.
3. 在 **"已安装的扩展"** 页上，找到要删除的扩展，然后选择它旁边的" **删除** "。
4. 在确认对话框中， **选择删除**。
