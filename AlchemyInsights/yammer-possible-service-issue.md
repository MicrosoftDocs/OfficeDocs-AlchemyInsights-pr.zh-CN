# <a name="yammer---possible-service-issue"></a>Yammer-可能的服务问题

可能存在暂时性的问题，即用户在访问 Yammer 时可能会遇到错误。 检查 [服务运行状况仪表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) 以查看这是否会影响你的网络。
