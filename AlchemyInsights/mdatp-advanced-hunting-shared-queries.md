# <a name="sharing-advanced-hunting-queries"></a>共享高级搜寻查询

高级的搜寻查询可在同一组织中的用户之间共享。 还可以查找在 GitHub 上公开共享的查询。 借助这些查询，你可以快速追寻特定威胁搜寻方案，而无需从头开始编写查询。
  
有关详细信息，请参阅 [在高级搜寻中使用共享查询](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/advanced-hunting-shared-queries)。
