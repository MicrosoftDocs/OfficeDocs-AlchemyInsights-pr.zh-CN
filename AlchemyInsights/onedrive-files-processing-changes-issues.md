# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive：疑难解答 "处理更改"

如果您在长时间内看到 "正在处理更改"，可能是由于打开了联机文件、队列中有多个文件或同步了大型文件，请参阅 [OneDrive：疑难解答 "处理更改"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) 以帮助解决此问题。

如果您的文件随 Outlook 一样 (更改，也会看到此消息。PST 文件或数据库) 。 了解如何将你经常使用的 [Outlook 数据文件](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) 从 OneDrive 中移出。
