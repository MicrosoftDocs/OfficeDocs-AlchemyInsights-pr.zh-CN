# <a name="resolving-issue-with-unknown-user-in-teams-chat"></a>解决团队聊天中的 "未知用户" 问题

有时，已删除的用户将显示为 "未知用户"。 这是一个 [已知问题](https://docs.microsoft.com/microsoftteams/troubleshoot/known-issues/removed-user-appears-as-unknown)。

如果您永久看到在团队聊天中显示为 "未知用户" 的用户，请尝试清除缓存：

1.  右键单击任务栏中的 "团队" 图标。 单击 "  **退出** "。
2.  浏览到计算机上的 %appdata%\Microsoft\teams\ 文件夹，然后删除该目录中的所有文件。

您可以通过确保匿名用户在大厅中等待，来阻止这些用户加入会议。 有关详细信息，请参阅 [更改团队会议的参与者设置](https://support.microsoft.com/office/change-participant-settings-for-a-teams-meeting-53261366-dbd5-45f9-aae9-a70e6354f88e)。
