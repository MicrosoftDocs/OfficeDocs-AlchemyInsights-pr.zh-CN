# <a name="client-context---no-enrolled-devices"></a>客户端上下文-没有已注册的设备

你的租户没有使用 Intune 的移动设备管理（MDM）注册设备。 如果要通过 Intune MDM 注册和管理设备，请参阅[此处](https://docs.microsoft.com/intune/device-enrollment)的文档以获取所需的平台。 如果您使用的是第三方 MDM 或仅应用程序保护功能，则不需要执行任何操作。
