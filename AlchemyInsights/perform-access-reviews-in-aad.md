# <a name="perform-access-reviews-in-azure-active-directory"></a>在 Azure Active Directory 中执行访问评审

指定的审阅者可以执行组成员身份、应用程序访问和角色分配的 Azure Active Directory 访问评审。 这些评论可帮助降低当用户具有多于其需要的访问权限时组织可能面临的安全风险。 可以使用"我的应用程序"或"我的访问 ["](https://go.microsoft.com/fwlink/?linkid=2134605) 执行访问 [评审](https://go.microsoft.com/fwlink/?linkid=2134505)。
