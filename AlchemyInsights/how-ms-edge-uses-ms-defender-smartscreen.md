# <a name="how-microsoft-edge-uses-microsoft-defender-smartscreen"></a>Microsoft Edge 如何使用 Microsoft Defender SmartScreen

为了使您在 web 上保持安全，Microsoft Edge 使用 Microsoft Defender SmartScreen 作为早期警告系统，针对 perpetrate 网络钓鱼攻击并通过集中式攻击分发恶意软件的网站。

管理员可以使用组策略、Microsoft Intune 或移动设备管理 (MDM) 设置来配置 Microsoft Defender SmartScreen。 根据您设置 Microsoft Defender SmartScreen 的方式，您可以向用户显示一个警告页面，并让他们继续转到网站或完全阻止网站。

若要了解详细信息，请参阅 microsoft [Edge 支持的 Microsoft Defender SmartScreen](https://go.microsoft.com/fwlink/?linkid=2133081) 和 [Microsoft Defender 应用程序防护](https://go.microsoft.com/fwlink/?linkid=2132839)。
