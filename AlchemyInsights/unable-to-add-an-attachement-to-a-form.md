# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>无法将附件添加到面向外部的窗体

如果您尝试将附件添加到外部 Microsoft Forms 调查中，则该功能尚不可用。 若要跟踪该功能请求的状态，请转到 [此 UserVoice 条目](https://go.microsoft.com/fwlink/?linkid=2133069)，并添加投票。
