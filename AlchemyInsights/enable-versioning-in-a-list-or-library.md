# <a name="enable-versioning-for-a-list-or-library"></a>为列表或库启用版本控制

1. 转到列表或库。
    
2. 单击 "**设置**"，然后单击 "**库设置**" 或 "**列表设置**"。
    
3. 单击 "**版本控制设置**"。
    
4. 选择所需的设置，然后单击 **"确定"**。
