# <a name="stuck-on-processing-order"></a>卡在处理订单上

如果已选择升级 Microsoft 365 订阅，但该订单已卡在“处理订单”中，请参阅[服务运行状况仪表板](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/servicehealth)发布 **SP207374**。 升级租户的过程将需要预配，这可能会延迟。 我们预计该过程将在 24 小时内完成；然而，我们已经看到很少的情况下，这些请求需要长达 48 小时。 如果订单处理时间超过 48 小时，请打开服务请求。
