# <a name="duplicate-object-synchronization-errors"></a>重复的对象同步错误

如果在同步期间收到以下错误之一:
  
- 您的 Microsoft Online Services 目录中已存在具有相同代理地址的同步对象。
    
- UserPrincipalName 的类似消息。
    
使用新的 AADConnect 运行状况诊断来解决重复的对象错误。 有关详细信息, 请参阅[诊断和修正重复的属性同步错误](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors)。
