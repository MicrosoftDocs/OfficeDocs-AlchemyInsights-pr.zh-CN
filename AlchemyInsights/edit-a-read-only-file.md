# <a name="edit-a-read-only-file"></a>编辑只读文件

在文档库中, 当您单击文档的名称时, Office 程序可能会意外地将其作为只读文件打开。 若要将其打开以进行编辑, 请单击 "**打开**" (三个点), 然后单击 "**编辑"。**
