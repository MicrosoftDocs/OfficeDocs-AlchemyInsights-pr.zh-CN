# <a name="change-your-account-picture"></a>更改您的帐户图片

1. 转到[您的 Microsoft 帐户信息](https://go.microsoft.com/fwlink/p/?linkid=860841)。
    
2. 选择**更改图片**。 
    
3. 选择**新的图片**。 
    
4. 选择您想要使用并选择**打开**的图片。 
    
5. 拖动或调整大小图像框并选择**保存**。 
    
请等待 24 小时对您的配置文件照片或名称生效的更改。
