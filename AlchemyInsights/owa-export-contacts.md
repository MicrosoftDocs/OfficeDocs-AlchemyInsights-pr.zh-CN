# <a name="export-contacts"></a>导出联系人

1. 在页面底部, 选择 "人员" 图标以转到 "人员" 页面。
2. 在工具栏上, 选择 "**管理** > **导出联系人**"。 
3. 选择从特定文件夹中导出所有联系人或仅导出联系人, 然后选择 "**导出**"。
4. 在页面底部, 选择 "**保存**" 以在默认的 "下载" 文件夹中保存 "contacts"。 或者, 选择 "**保存**" 旁边的箭头, 然后选择 "**另存为**"。

若要了解详细信息, 请参阅[从 Outlook 导出联系人](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web)。
