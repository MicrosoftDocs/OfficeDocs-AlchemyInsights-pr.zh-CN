# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>阻止必应中的 Microsoft 搜索与 Microsoft 365 应用一起安装

若要阻止必应 Microsoft 搜索的后台服务随 Microsoft 365 企业应用版一起安装，请使用以下选项之一：

- Office 部署工具或组策略
- Microsoft Endpoint Configuration Manager (当前分支) 或 Microsoft Intune

若要了解更多信息，请参阅如何阻止必应中的 Microsoft 搜索的后台服务与 [Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2151946)企业应用版一起安装。
