# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>移动设备上使用共享的邮箱

从移动设备安装的 Outlook 应用程序，您无法访问共享的邮箱。此功能吗？让 Outlook 工程团队知道。[此处投票](https://go.microsoft.com/fwlink/?linked=862116)！
  
仅可以通过打开浏览器中，登录到 Office 365，然后转到 web 上的 Outlook 移动设备上使用共享的邮箱。有关说明，请参阅[Add 中 Outlook web 上共享邮箱](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)。
  
如果必须访问共享的邮箱移动设备上使用 Outlook 应用程序，请考虑创建 Office 365 组而不是使用共享的邮箱。有关详细信息，请参阅[比较组](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)。
  
了解详细信息：[获取为 Outlook for iOS 和 Android 应用程序内帮助](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
