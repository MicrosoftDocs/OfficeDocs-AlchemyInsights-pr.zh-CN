- [当用户获取共享链接时，更改默认链接类型](https://docs.microsoft.com/sharepoint/change-default-sharing-link)（当用户获取链接进行共享时更改默认链接类型）

- [停止共享 OneDrive 或 SharePoint 文件或文件夹，或更改权限](https://support.office.com/article/stop-sharing-onedrive-or-sharepoint-files-or-folders-or-change-permissions-0a36470f-d7fe-40a0-bd74-0ac6c1e13323)

- [共享和权限](https://support.office.com/article/Sharing-and-permissions-ac85fbf1-2431-49bf-8690-f1a2b98af65f#ID0EAABAAA=Manage_permissions)

- [为 Sharepoint Online 启用或禁用外部共享](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off)
