# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>设置转移到多个电子邮件地址

首先，导航到[管理组](https://portal.office.com/adminportal/home#/groups)
  
1. 创建*通讯组列表* 
    
2. 向其添加多个用户
    
3. 然后配置邮箱转发到的组
