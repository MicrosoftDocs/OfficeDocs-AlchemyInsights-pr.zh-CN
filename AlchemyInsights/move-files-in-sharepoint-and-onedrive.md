# <a name="move-files-in-sharepoint-and-onedrive"></a>在 SharePoint 和 OneDrive 中移动文件

您可以将文件从 web 浏览器本身移动到 SharePoint 和 OneDrive 中的不同位置。 请参阅 [移动或复制文件](https://support.microsoft.com/office/move-or-copy-files-in-sharepoint-00e2f483-4df3-46be-a861-1f5f0c1a87bc?ui=en-US&rs=en-US&ad=US)。


> [!NOTE]
> 如果使用的是 SharePoint Online 的经典体验，"**移动到**" 将不可用。
