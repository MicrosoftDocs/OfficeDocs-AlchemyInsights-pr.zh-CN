# <a name="working-with-sharepoint-permission-levels"></a>使用 SharePoint 权限级别

SharePoint 附带了您可以使用的默认权限级别，例如 **"参与"** 和"**仅查看"。** 但是，如果其中一个不符合你的需求，你可以创建新的权限级别。 请记住，如果不更改默认值，跟踪权限级别将更容易。

[如何创建和编辑权限级别](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
