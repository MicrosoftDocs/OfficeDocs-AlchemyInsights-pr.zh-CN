# <a name="email-forwarding-report"></a>电子邮件转发报告

若要查看关于组织中有多少自动转发邮件的报告，请查看 [Microsoft 365 安全与合规中心](https://protection.office.com/#/homepage)内的[邮件流见解](https://docs.microsoft.com//office365/securitycompliance/mail-flow-insights-v2)。
  
请注意：如果这是你首次登录安全 &amp; 合规中心，则可能需要在几分钟后刷新浏览器以查看所有可用选项。
