# <a name="admin-roles"></a>管理员角色

- 在管理中心中选择“[角色](https://admin.microsoft.com/Adminportal/Home#/roles)”以分配或了解有关管理员角色的更多信息。

- 然后选择一个或多个角色名称，然后单击“**分配角色**”，或单击“**比较角色**”以了解更多信息。
