# <a name="use-your-browser-to-check-spelling"></a>使用浏览器检查拼写

Outlook.com 中没有拼写检查选项。 若要检查拼写, 您需要使用 web 浏览器。 在 Microsoft Edge、Internet Explorer 10 及更高版本以及 Firefox、Chrome 和 Safari 的当前版本中均提供拼写检查。 检查 web 浏览器的选项以了解有关如何检查拼写的详细信息。
