# <a name="how-to-get-support-in-outlookcom"></a>如何在 Outlook.com 中获取支持

抱歉, Outlook.com 不提供电话支持, 但我们确实拥有可帮助你的专用团队。
在上面的 "**帮助**" 框中输入您的问题或说明你的问题, 然后选择 "**获取帮助**"。
