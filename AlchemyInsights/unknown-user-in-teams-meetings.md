# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>工作组会议或聊天中出现 "未知用户"

用户可能已从 Microsoft 365 中删除。 在这种情况下，请尝试从团队中删除用户。  

尝试在团队关闭时清除以下缓存，然后重新启动团队： `C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

有关此问题的更新，请参阅  [Microsoft 团队中的已删除用户在 "未知用户" 中显示为 "未知用户"](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown)。
