# <a name="public-folder-access-from-mobile-devices"></a>从移动设备的公用文件夹访问

公用文件夹不是可从移动设备访问的。
  
如果您想要已功能类似于公用文件夹，并从移动设备访问它，请参阅 for 更好的替代项的[了解有关 Office 365 组](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)。
