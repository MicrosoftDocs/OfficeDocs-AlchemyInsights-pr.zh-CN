# <a name="disabling-advanced-outlookcom-security"></a>禁用高级 Outlook.com 安全性

若要为您的帐户提供最佳保护, Outlook.com 将检查邮件中的任何链接是否为网络钓鱼恶作剧, 或者是否有可能将病毒或恶意软件下载到您的计算机上。 如果你确信要删除此高级 Outlook.com 安全, 请在下面单击 **"是"** 。
