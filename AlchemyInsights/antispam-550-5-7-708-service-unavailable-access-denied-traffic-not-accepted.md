# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 访问被拒绝, 不接受此 IP 流量的流量

当从已知的低信誉 IP 地址发送电子邮件时, 新客户通常使用此错误。

如果您必须在购买许可证之前从这些 IP 地址发送电子邮件, 请联系支持人员以请求例外, 直到您能够购买许可证。
