# <a name="enter-imap-port-numbers"></a>输入 IMAP 端口号

通过 "预先安装" 屏幕登录，并在冒号的后面添加端口详细信息，并将传入和传出服务器地址添加到行中。  (可以从您的电子邮件服务提供商获取这些详细信息。 )  

转到 "**设置**"  >  "**添加帐户**  >  **IMAP** >"，然后单击 "**显示高级设置**"，然后输入详细信息。 

*传入邮件服务器*： imap.servername.com:000 (通常为993或 143)  

*传出邮件服务器*： smtp.servername.com:000 (通常为465或 25)  

请注意，冒号很重要。
