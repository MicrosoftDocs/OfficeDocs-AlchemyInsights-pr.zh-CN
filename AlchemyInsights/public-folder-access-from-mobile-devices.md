# <a name="public-folder-access-from-mobile-devices"></a>从移动设备访问公用文件夹

无法使用移动设备访问公用文件夹，请使用 OWA (Outlook on Web) 或 Outlook for Mac 客户端访问公用文件夹。

如果您希望功能与公用文件夹类似，并且在移动设备上具有可访问的内容，请参阅 [了解 Microsoft 365 组](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) 以获得更好的替代方法。
