# <a name="microsoft-viva-learning-availability"></a>Microsoft Viva 学习可用性

从 2021 年 2 月开始，我们将与少量客户和合作伙伴一起预览适用于 Teams 的 Viva 学习应用，并预期它将在晚些时候正式发布。 我们鼓励你 [在此处注册](https://aka.ms/VivaLearningSignup) ，以了解有关 Viva Learning 的详细信息，包括新体验可用时的进一步详细信息。

有关详细信息，请参阅 [宣布推出 Microsoft Viva 学习私人预览版](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/announcing-microsoft-viva-learning-private-preview/ba-p/2107023)。
