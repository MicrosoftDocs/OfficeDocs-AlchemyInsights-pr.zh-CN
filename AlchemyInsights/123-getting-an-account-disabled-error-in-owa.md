# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>获取帐户禁用在 web 上的 Outlook 中的错误？

如果您收到**您的帐户已被禁用**错误与`X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException`当您尝试打开 Outlook web （之前被称为 Outlook Web App） 上的，您组织的管理员可能会禁用您对 web 上的 Outlook。您的管理员可以通过[启用或禁用邮箱的 Outlook Web App](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)的主题中的步骤来解决此错误。
