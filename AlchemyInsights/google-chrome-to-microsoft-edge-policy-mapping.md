# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>了解 Google Chrome 策略和 Microsoft Edge 策略之间的映射

Google Chrome 策略映射到浏览器版本 80 中支持的相关 Microsoft Edge 策略。 若要了解更多信息，请参阅 [Google Chrome 到 Microsoft Edge 策略映射](https://go.microsoft.com/fwlink/?linkid=2141933)。
