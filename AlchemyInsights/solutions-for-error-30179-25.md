# <a name="solutions-for-error-30179-25"></a>解决方案的错误 30179 25

您可能会看到错误消息"无法 stream Office"（错误代码[30179 25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)） 如果您的硬盘空间不足。 
  
[您的硬盘上的空间忙](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space)，然后[重新安装 Office](https://portal.office.com/OLS/MySoftware.aspx)。
