# <a name="solutions-for-error-30179-25"></a>错误30179-25 的解决方案

如果你的硬盘空间不足, 则可能会看到消息 "无法流式处理 Office" (错误代码[30179-25](https://support.office.com/article/e40d3c7d-98f6-4284-94a0-882beaa44593?wt.mc_id=Alchemy_ClientDIA)) 的错误。 
  
[释放硬驱上的空间](https://support.microsoft.com/help/12425/windows-10-free-up-drive-space), 然后[重新安装 Office](https://portal.office.com/OLS/MySoftware.aspx)。
