# <a name="change-permissions-for-a-sharepoint-list-or-library"></a>更改 SharePoint 列表或库的权限

1. 转到 web 浏览器中的 SharePoint 网站并打开列表或库。
    
2. 单击右上角，设置图标，然后单击**库设置**或**列表设置**。（如果您看不到该命令，选择**库**或**列表**选项卡以打开功能区，然后单击功能区上的**库设置**或**列表设置**。） 
    
3. 单击**权限和管理**下的**此列表的权限**或**此文档库的权限**。
    
请注意，默认情况下，SharePoint 列表和库从网站中继承其权限。如果中断权限继承，当前权限将复制到列表或库中的所有内容。如果稍后更改网站的权限，所做的更改不会应用到列表或库中的项目。
