# <a name="skype-icon-missing"></a>缺少 Skype 图标

我们已暂时将 Skype 图标从 Outlook.com 中删除作为设计刷新和功能更新的一部分，我们非常兴奋要推出。我们感谢你的耐心等待，我们为你准备新的和改进的体验。

您仍可以通过浏览器在 [web.skype.com](https://web.skype.com)中使用 Skype。
