# <a name="map-a-sharepoint-library-to-a-network-drive"></a>将 SharePoint 库映射到网络驱动器

将库映射为网络驱动器是临时的，并且仅通过 Internet Explorer 支持。 您必须偶尔在 Internet Explorer 中打开 SharePoint 网站，并选择 **"保持登录**状态" 以阻止会话过期。 而是[使用新的 OneDrive 同步客户端同步 SharePoint 文件](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)</a> ，该客户端[根据需要提供文件](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e)。 在不使用本地存储空间的情况下访问 OneDrive 中的所有文件。

如果选择映射驱动器，而不是[使用新的 OneDrive 同步客户端](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)，请确保按照以下文章中的步骤操作。 


**如何配置映射的网络驱动器并对其进行疑难解答**


请参阅[疑难解答连接到 SharePoint Onlines 的映射网络驱动器](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)。

注意：如果在 Windows 8 或 Windows 7 中使用 Internet Explorer 10，并且在映射驱动器时接收**访问被拒绝**或**路径不可访问**，请安装[此修补程序](https://support.microsoft.com/help/2846960)以解决此问题。
