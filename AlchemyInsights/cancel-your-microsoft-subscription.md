# <a name="cancel-your-microsoft-subscription"></a>取消 Microsoft 订阅

1. 转到[服务与订阅](https://account.microsoft.com/services/)，使用购买订阅时所用的 Microsoft 帐号登录。

2. 查找订阅，然后单击“**管理**”。 如果看到“**启用定期账单**”，而非“**管理**”，则请跳到“疑难解答”部分; 否则，请继续执行步骤 3。

3. 在下一页上，单击“**取消**”或“**升级或取消**”，具体取决于你的订阅类型。

4. 按照页面上的说明进行操作，继续取消操作。
