# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>使用 Jamf 在 macOS 上配置 Microsoft Edge 策略设置

若要在 macOS (Microsoft Edge) Stable 渠道 81 或更高版本，请使用 Jamf Pro 10.19 上的 Microsoft Edge 策略清单文件。

有关详细信息，请参阅使用 [Jamf](https://go.microsoft.com/fwlink/?linkid=2134761)在 macOS 上配置 Microsoft Edge 策略设置。
