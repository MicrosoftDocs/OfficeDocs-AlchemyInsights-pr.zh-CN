# <a name="add-phone-number"></a>添加电话号码

在可以将组织中的用户设置为拨打和接听电话前，必须为其[获取电话号码](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)。 我们提供了完成电话系统设置的分步过程。 请查看[设置组织内的电话系统](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)。
