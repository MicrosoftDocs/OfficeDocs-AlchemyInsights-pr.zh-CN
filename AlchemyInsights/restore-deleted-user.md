# <a name="restore-a-user-in-office-365"></a>在 Office 365 中还原用户

还原使用这些步骤的 Office 365 中的用户：
  
1. 转到[用户\>已删除的用户](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/deletedusers)。
    
2. 选择用户，然后选择**还原**。
    
3. 按照提示设置密码。
    
4. 单击**发送电子邮件并关闭**并完成 ！
    
不是这么简单吗？有关详细信息和步骤屏幕截图，签出此文章：[还原 Office 365 中的用户](https://support.office.com/article/https://support.office.com/en-us/article/Restore-a-user-in-Office-365-2c261e42-5dd1-48b0-845f-2a016d29cfc1.aspx)
