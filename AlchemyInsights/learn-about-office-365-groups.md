# <a name="learn-about-microsoft-365-groups"></a>了解 Microsoft 365 组

Microsoft 365 Groups 是一项服务，可与您使用的 Office 365 工具配合工作，以便在撰写文档、创建电子表格、处理项目计划、安排会议或发送电子邮件时，与您的团队协作。 有关详细信息，请参阅[了解 Microsoft 365 组](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)。
