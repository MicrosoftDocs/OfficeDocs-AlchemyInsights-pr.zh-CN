# <a name="profile-property-synchronization"></a>配置文件属性同步

无法映射 AD 导入的其他属性以从 Azure Active Directory 同步到用户配置文件应用程序。 AD 导入仅限于预配置的一组属性，以确保计时器作业的一致性能。
  
[了解有关同步的配置文件属性的详细信息](https://go.microsoft.com/fwlink/?linkid=875671)
