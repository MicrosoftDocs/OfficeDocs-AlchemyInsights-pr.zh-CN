# <a name="print-an-attachment"></a>打印附件

若要打印电子邮件附件，下载并将附件保存到本地计算机或 OneDrive 然后用适当的程序打开它们。
  
1. 打开包含您想要打印的附件的邮件。
    
2. 下面附件，选择**下载**。 
    
3. 已下载文件后，将其与相应的程序中打开并选择打印选项。例如，在 Word 中打开一个.doc 文件，然后打印。
    
有关详细信息，阅读[打印电子邮件和附件 Outlook.com 中的](https://go.microsoft.com/fwlink/?linkid=2021110&amp;clcid=0x409)。
