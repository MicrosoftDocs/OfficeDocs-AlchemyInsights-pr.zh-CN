# <a name="restore-deleted-files-or-folders-in-sharepoint"></a>在 SharePoint 中还原已删除的文件或文件夹

如果意外删除了 SharePoint 中的文件或文件夹，则稍后可能会从回收站中恢复该文件或文件夹。

* [在 SharePoint 中还原已删除的文件或文件夹](https://support.microsoft.com/office/restore-items-in-the-recycle-bin-that-were-deleted-from-sharepoint-or-teams-6df466b6-55f2-4898-8d6e-c0dff851a0be)

如果大量 SharePoint 文件被恶意软件删除、覆盖、损坏或感染，您可以将整个 SharePoint 库还原到上一次。

* [还原文档库](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a)
