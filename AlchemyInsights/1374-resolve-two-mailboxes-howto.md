# <a name="a-user-has-two-mailboxes"></a>用户具有两个邮箱

使用 Azure Active Directory 连接 （AAD 连接） 或 DirSync 的混合环境可能意外导致用户具有两个邮箱： 一个内部部署组织和一个云中中。
  
若要解决此问题，请参阅[永久清除以前邮箱信息](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/)。
