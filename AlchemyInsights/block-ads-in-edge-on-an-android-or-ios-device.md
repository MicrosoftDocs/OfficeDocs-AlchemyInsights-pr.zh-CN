# <a name="block-ads-in-the-microsoft-edge-browser-on-an-android-or-ios-device"></a>阻止 Android 或 iOS 设备上 Microsoft Edge 浏览器中的广告

阻止除 AdBlock Plus 认为可以接受广告之外的所有广告:
1. 选择 **…** > **(设置和更多)** > **设置** > **内容拦截器**。
2. 打开 **“阻止广告”**。

阻止所有广告:
1. 选择 **…** > **(设置和更多)** > **设置** > **内容拦截器**。
2. 打开 **“阻止广告”**。
3. 选择 **“高级设置”**。
4. 关闭 **“允许可接受的广告”**。
