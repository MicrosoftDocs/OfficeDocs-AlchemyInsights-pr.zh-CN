# <a name="enable-microsoft-stream"></a>启用 Microsoft Stream

除非帐户获得许可，否则组织员工无法使用 Sream 服务。 你可以为它们分配 Stream 许可证，这样每个用户就无需手动注册此服务了。

- [为组织中的每个人启用 Microsoft Stream](https://docs.microsoft.com/stream/assign-user-licenses)。
