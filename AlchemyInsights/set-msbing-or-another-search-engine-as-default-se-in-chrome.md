# <a name="set-microsoft-bing-or-another-search-engine-as-the-default-search-engine-in-google-chrome"></a>在 Google Chrome 中将 Microsoft 必应或其他搜索引擎设置为默认搜索引擎

若要在 Google Chrome 中设置默认搜索引擎：

1. 打开 Google Chrome。
1. 在右上角，选择显示三个垂直点的图标， **然后选择设置**。
1. 在 **"搜索引擎"** 下，在地址栏中使用的搜索引擎旁边，选择向下箭头，然后选择 **"必应**"或其他搜索引擎。

切换到除必应外的其他搜索引擎不会删除必应扩展，因此你仍然可以通过扩展驱动的飞出框轻松搜索与工作相关的内容。
