# <a name="copy-and-paste-in-outlookcom"></a>在 Outlook.com 中复制和粘贴

Outlook.com 中没有 "复制" 或 "粘贴" 按钮, 但您可以使用键盘快捷方式:

**在电脑上:** CTRL + C 复制, 按 CTRL + X 进行剪切, 按 CTRL + V 进行粘贴。

**在 Mac 上:** CMD + C 复制、CMD + X 进行剪切以及 CMD + V 粘贴。
