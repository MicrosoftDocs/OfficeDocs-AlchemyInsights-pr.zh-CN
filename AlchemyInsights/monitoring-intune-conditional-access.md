# <a name="troubleshoot-issues-with-microsoft-intune"></a>解决 Microsoft Intune 的问题

查看下列来解决此问题现在资源。
  
1. 使用[Intune 疑难解答门户](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)诊断和解决一些常见的故障。查看[此文档](https://docs.microsoft.com/intune/help-desk-operators)的详细信息。
    
2. 查看[Intune TechNet](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)查找答案和常见的问题的解决方案。
    
3. 签出[服务运行状况](https://portal.office.com/AdminPortal/Home#/servicehealth)查看服务状态。 
    
4. 查看[Intune 文档](https://docs.microsoft.com/intune/)规划、 操作方法，疑难解答文章和更多内容。
