# <a name="delete-a-contact"></a>删除联系人

1. 选择屏幕左下角的 "**人员**" 图标。

2. 选择要删除的联系人, 然后选择 "**删除**"。

    如果不能选择 "**删除**", 则可能是联系人来自 Skype 或已连接的社交网络帐户。 若要删除联系人, 请转到 Skype 或联系人所在的帐户。

3. 选择 "**删除**" 以确认。
