# <a name="about-room-mailboxes"></a>关于会议室邮箱

会议室邮箱是分配给物理位置（例如会议室、礼堂或培训室）的资源邮箱。 有关会议室邮箱的详细信息，请参阅[创建和管理会议室邮箱](https://go.microsoft.com/fwlink/p/?linkid=717533)。
