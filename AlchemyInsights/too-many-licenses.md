# <a name="too-many-licenses"></a>太多许可证？

如果您认为您正在收取太多许可证，这通常是由于具有购买更多许可证，则需要。
  
 **减少购买的许可证数**
  
1. 在管理中心，转到[订阅](https://go.microsoft.com/fwlink/p/?linkid=842054)页面，或选择" **帐单**"\>" **订阅**"。
    
2. 在左侧，选择要从中删除许可证的订阅。
    
3. 选择" **添加/删除许可证**"。
    
4. 输入所需的用户总数。例如，如果当前 25 个许可证，并且您想要删除 5，在此字段中输入 20。
    
5. 选择" **提交**"。
