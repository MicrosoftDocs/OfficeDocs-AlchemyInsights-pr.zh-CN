# <a name="cant-select-contacts-when-composing-email"></a>撰写电子邮件时无法选择联系人

撰写新电子邮件时, 某些客户单击 "**收件人**" 或 **"抄送"** 按钮时不会看到他们的联系人列表。 若要选择联系人, 请在 "**收件人**" 或 **"抄送**" 字段中键入, 将显示您的相关联系人。
