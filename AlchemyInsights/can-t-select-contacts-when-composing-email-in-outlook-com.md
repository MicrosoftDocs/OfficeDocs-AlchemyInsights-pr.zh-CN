# <a name="cant-select-contacts-when-composing-email"></a>撰写电子邮件时无法选择联系人

在撰写新电子邮件并单击 "**收件人**" 或 **"抄送"** 按钮时, 某些客户可能看不到其联系人列表。 若要选择名称, 请在 "**收件人**" 或 **"抄送**" 字段中键入, 并显示您的相关联系人。
