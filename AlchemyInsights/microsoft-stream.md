# <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream 是一种企业视频服务，组织内部的人员可安全地在其中上传、查看和共享视频。 

有关详细信息，请参阅：

- [什么是 Microsoft Stream？](https://docs.microsoft.com/stream/overview)
- [Office 365 视频转换为 Microsoft Stream](https://docs.microsoft.com/stream/migrate-from-office-365)
- [Stream Service 更新](https://techcommunity.microsoft.com/t5/microsoft-stream-service-updates/bd-p/StreamAnnouncements)
- [Microsoft Stream 中的组和通道](https://docs.microsoft.com/stream/groups-channels-organization)
- [Microsoft Stream 中的实时事件](https://docs.microsoft.com/stream/live-event-overview)
- [在 Microsoft Stream 中共享你的视频](https://docs.microsoft.com/stream/portal-share-video)
