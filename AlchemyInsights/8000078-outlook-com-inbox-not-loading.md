# <a name="inbox-not-loading-with-firefox"></a>不使用 Firefox 加载收件箱

Firefox 浏览器中的某些加载项可能会阻止您的 Outlook.com 收件箱加载。
  
若要了解如何[禁用或删除加载项](https://support.mozilla.org/kb/disable-or-remove-add-ons), 请访问 Firefox 支持。
