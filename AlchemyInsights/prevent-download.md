# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>阻止从 SharePoint 或 OneDrive 下载文件

有多种方法可用于阻止或阻止下载发生。 有关相应的方法, 请访问以下链接:

- [使用 Azure AD 条件访问阻止文件下载](https://docs.microsoft.com/en-us/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [通过非托管设备控制访问](https://docs.microsoft.com/en-us/sharepoint/control-access-from-unmanaged-devices)

- [在 SharePoint 文档库上配置 IRM 策略以限制内容的下载](https://docs.microsoft.com/en-us/office365/securitycompliance/set-up-irm-in-sp-admin-center)
