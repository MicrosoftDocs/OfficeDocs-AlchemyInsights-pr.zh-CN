# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>删除经典 SharePoint 团队网站或子网站

1. 转到网站。
    
2. 单击右上角的 "**设置**" 图标。 
    
3. 单击 "**网站设置**"。 在 "**网站操作**" 下, 单击 "**删除此网站**", 然后单击 "**删除**"。
