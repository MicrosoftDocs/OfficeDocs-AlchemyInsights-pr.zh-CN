# <a name="supported-version-of-the-sharepoint-migration-tool"></a>受支持的版本的 SharePoint 迁移工具

当前 SharePoint 迁移工具仅支持从 SharePoint 2013 的迁移。目前不支持版本 2010年和 2016年。
  
有关详细信息，请参阅[简介 SharePoint 迁移工具](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)。
