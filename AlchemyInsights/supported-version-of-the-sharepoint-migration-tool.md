# <a name="supported-version-of-the-sharepoint-migration-tool"></a>SharePoint 迁移工具的受支持版本



目前, sharepoint 迁移工具仅支持从 SharePoint 2013 迁移。 目前不支持版本2010和2016。
  
有关详细信息, 请参阅[简介 SharePoint 迁移工具](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)。
