# <a name="calendar-publishing-information"></a>日历发布信息

- Microsoft 365 管理员可以为组织启用或禁用日历发布和日历共享。 有关详细信息，请参阅[启用日历发布，以便用户可以](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)[与外部用户](https://docs.microsoft.com/microsoft-365/admin/manage/share-calendars-with-external-users)共享信息和共享日历。

- Outlook 用户可以共享自己的日历。 有关详细信息，请参阅[与其他人共享 Outlook 日历](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)。
