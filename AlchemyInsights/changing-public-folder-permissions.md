# <a name="changing-public-folder-permissions"></a>更改公用文件夹权限

您和您的用户可以更改 Outlook 中的公用文件夹权限。 请参阅[设置或更改公用文件夹的权限](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005)。 管理员还可以更改 Exchange 管理中心中的权限:
  
- 在 Microsoft 365 管理中心, 转到 "**管理员中心** \> **Exchange**"。
    
- 选择 "**公用文件夹**"。
    
在这里, 可以通过向权限分配安全组的用户来更改单个公用文件夹的权限。 若要更改公用文件夹的权限, 您需要是该文件夹的所有者。
