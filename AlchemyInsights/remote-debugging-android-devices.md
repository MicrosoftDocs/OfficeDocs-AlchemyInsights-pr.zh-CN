# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>使用 Microsoft Edge 从开发计算机远程调试 Android 设备

若要从计算机远程调试 Android 设备上的实时内容，请执行以下操作：

1. 设置 Android 设备进行远程调试，并在开发计算机上进行发现。
2. 在开发计算机上，检查并调试 Android 设备上的实时内容。
3. 将实时内容从 Android 设备截屏视频到开发计算机上的 DevTools 实例。
