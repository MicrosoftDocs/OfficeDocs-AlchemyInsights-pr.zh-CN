# <a name="upload-process-overview-in-microsoft-stream"></a>Upload Microsoft Stream 中的流程概述

每次上传视频时，Microsoft Stream 都会自动将其编码为多个比特率，以便视频可以使用自适应比特率流式传输。 Microsoft Stream 根据观看者可用的带宽优化播放视频的比特率和质量。
