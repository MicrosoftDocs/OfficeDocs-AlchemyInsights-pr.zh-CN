# <a name="upload-process-overview-in-microsoft-stream"></a>在 Microsoft Stream 中上载过程概述

无论何时上传视频，Microsoft Stream 都会自动将其编码到多个 bitrates 中，以便可以使用自适应比特率数据流对视频进行流式处理。 Microsoft Stream 根据查看器的可用带宽来优化播放视频的比特率和质量。

- [在 Microsoft Stream 中上载过程概述](https://docs.microsoft.com/stream/upload-process-overview)
