# <a name="set-up-one-monitor"></a>设置一个监视器

**检查电脑的项目设置**

1. 同时按以下键打开项目设置： Windows 键（Windows 徽标 "开始" 按钮） + P。
2. 根据你的偏好，选择 "**复制**"、"**扩展**" 或 "**仅限第二个屏幕**"。

**注意：** 如果你的电脑仅有一个显示端口（如 HDMI 或 VGA），则需要一个坞站或 USB 适配器来设置多台监视器。

此外，如果你可以设置一个监视器而不是多台监视器，你的电脑可能不会与多个监视器兼容。 有关详细信息，请查看电脑的原始文档。
