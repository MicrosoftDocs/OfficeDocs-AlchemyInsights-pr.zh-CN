# <a name="deploy-microsoft-365"></a>部署 Microsoft 365

[Microsoft 365 部署顾问](https://go.microsoft.com/fwlink/?linkid=2072646)为商业客户提供设置生产力工具、安全策略和设备管理功能的指南。 通过 Microsoft 365 商业高级版或 Microsoft 365 企业版订阅，你可以使用此顾问来设置和配置组织的设备。

你将收到指南并访问资源，以启用云服务、将设备更新到受支持的最新版本的 Windows 10，以及将设备加入到 Azure Active Directory (Azure AD) ，所有这些都位于一个中心位置。
