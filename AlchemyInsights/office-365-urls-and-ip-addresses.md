# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 Url 和 IP 地址范围

Microsoft 365 需要通过名为 "*终结点*" 的特定 IP 地址连接到 Internet。
有关需要访问的终结点的列表，请参阅[Microsoft 365 url 和 IP 地址范围](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Microsoft 365 IP 地址和 URL web 服务可帮助您更好地识别和区别 Microsoft 365 网络流量，使您更轻松地评估、配置和保持最新的更改。 请参阅[Office 365 IP 地址和 URL web 服务](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)。
