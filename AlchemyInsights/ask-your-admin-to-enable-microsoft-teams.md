# <a name="youre-missing-out-ask-your-admin-to-enable-microsoft-teams"></a>你缺少！ 要求管理员启用 Microsoft Teams

如果拥有 EDU 租户，需要手动启用 Microsoft Teams。 为此，请转到 Microsoft 365 管理中心 >Services  **和 Microsoft Teams**>外接程序。 如果使用的是新的 Teams 管理中心，必须关闭"试用新管理中心"以 ****   查找服务和 **加载项**。 

**重要** 提示：可能需要等待 24 小时，更改才能生效。
