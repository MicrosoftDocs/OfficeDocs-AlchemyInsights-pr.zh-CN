# <a name="youre-missing-out-ask-your-admin-to-enable-microsoft-teams"></a>你缺少！ 请求管理员启用 Microsoft 团队

如果拥有 EDU 租户，需要手动启用 Microsoft Teams。 为此，请转到 Microsoft 365 管理中心 > **Services 和外接程序 > Microsoft 团队**。 如果你使用的是新团队管理中心，则必须关闭 **"尝试新管理中心"**   以查找 **服务和外接程序**。 

**重要说明**：您可能需要等待24小时，更改才会生效。
