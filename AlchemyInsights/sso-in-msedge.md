# <a name="single-sign-on-sso-in-microsoft-edge"></a>在 Microsoft Edge (SSO) 单一登录

Microsoft Edge 支持以下类型的单一登录 (SSO) ：
- 基于 PRT 的 SSO
- 无缝 SSO
- 基于 WIA 的 SSO
