# <a name="domain-service-synchronization"></a>域服务同步

Azure Active Directory 域服务 (Azure AD DS) 托管域中的对象和凭据可以在域中本地创建，也可以从 Azure Active Directory (Azure AD) 租户同步。 首次部署 Azure AD DS 时，将配置并启动自动单向同步以从 Azure AD 复制对象。 此单向同步将继续在后台运行，使 Azure AD DS 托管域保持最新状态以及 Azure AD 的任何更改。 Azure AD DS 不会同步回 Azure AD。

有关 Azure Active Directory 域服务同步的更多详细信息，请参阅 [域服务同步](https://docs.microsoft.com/azure/active-directory-domain-services/synchronization)。
