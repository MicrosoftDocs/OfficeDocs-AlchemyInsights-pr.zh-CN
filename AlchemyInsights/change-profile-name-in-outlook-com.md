# <a name="change-your-profile-name"></a>更改您的配置文件名称

1. 转到[你的 Microsoft 帐户信息](https://go.microsoft.com/fwlink/p/?linkid=860841)。
    
2. 选择 "**编辑名称**"。 
    
3. 输入名字和姓氏, 然后选择 "**保存**"。 
    
请允许长达24小时对你的配置文件名称所做的更改生效。
  
若要了解详细信息, 请参阅[在 Outlook.com 中更改你的个人资料照片或名称](https://go.microsoft.com/fwlink/?linkid=873110)。
