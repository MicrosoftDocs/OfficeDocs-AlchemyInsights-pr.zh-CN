# <a name="cancel-power-bi"></a>取消Power BI

Power BI在管理中心Microsoft 365取消。

1. In the admin center， go to the **Billing > [Your products](https://go.microsoft.com/fwlink/p/?linkid=842054)** page.
2. 在“**产品**”选项卡上，找到要取消的订阅。 Select the three dots (more actions) ， then select **Cancel subscription**.
3. 在“**取消订阅**”窗格中，选择取消原因。 (可选)提供反馈。
4. 选择“**保存**”。

**退款：任何按比例计算退款将在下一个计费周期内退回给你。**

> [!NOTE]
> 如果看不到取消选项，请选择订阅以查看详细信息页面。 在"**订阅和付款设置"** 部分，选择"**取消订阅"。**

若要了解更多信息，请参阅 [取消订阅](/microsoft-365/commerce/subscriptions/cancel-your-subscription)。
