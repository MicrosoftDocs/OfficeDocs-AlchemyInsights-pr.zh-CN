# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>将 Windows 10 设备载入 Microsoft Defender 高级威胁防护 (MDATP) 

MDATP 支持以下适用于 Windows 10 的部署工具和方法：

- 组策略
- Microsoft Endpoint Configuration Manager
- 移动设备管理，包括 Microsoft Intune
- 本地脚本

若要了解更多信息，请参阅 Windows [10 设备的载入工具和方法](https://go.microsoft.com/fwlink/?linkid=2143460)。
