# <a name="set-up-and-configure-communication-compliance"></a>设置和配置通信合规性

为了供将来参考，受通信合规性监视的用户必须具有 Microsoft 365 E5 合规性许可证或 Office 365 企业版 E3 许可证。

* 如果要设置或配置通信合规性，请参阅通信 [合规性入门](https://go.microsoft.com/fwlink/?linkid=2111549)。
* 如果你没有现有的企业版 E5 计划，并且想要试用监督，请注册 [Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。
