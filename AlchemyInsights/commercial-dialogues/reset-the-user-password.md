# <a name="reset-the-users-password"></a>重置用户密码

您还需要重置用户密码，然后才能执行任何其他操作。 操作步骤如下：

1. 选择 **"用户**  >  **[活动用户"。](https://go.microsoft.com/fwlink/p/?linkid=834822)**
1. 选择帐户遭到入侵的用户。
1. 在出现的飞出中，单击"**重置密码"。**

> [!NOTE]
> 不要通过电子邮件将密码发送给用户。
