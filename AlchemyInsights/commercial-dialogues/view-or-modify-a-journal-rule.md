# <a name="view-or-modify-a-journal-rule"></a>查看或修改日记规则

操作步骤如下：

1. 在 [Exchange 管理中心 中](https://go.microsoft.com/fwlink/p/?linkid=2059104)，转到 **"合规性管理**  >  **""日记规则"。**
2. 在列表视图中，你将看到组织的所有日记规则。
3. 双击要查看或修改的规则。
4. 在 **日记规则** 中，修改你需要的设置。
