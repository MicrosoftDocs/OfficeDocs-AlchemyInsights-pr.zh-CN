# <a name="pop-and-imap4-settings"></a>POP 和 IMAP4 设置

若要找到您自己的邮箱的 POP3 和 IMAP4 服务器设置，请参阅[POP 和 IMAP 电子邮件 Outlook 设置](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx)。
  
管理员可以启用和配置内部部署 Exchange 组织中的 POP3 和 IMAP4。有关详细信息，请参阅[启用和配置 Exchange 服务器上的 POP3](https://technet.microsoft.com/library/bb124934.aspx)和[启用和配置 Exchange 服务器上的 IMAP4](https://technet.microsoft.com/library/bb124489.aspx)。
