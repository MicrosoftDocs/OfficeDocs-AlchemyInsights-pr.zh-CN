# <a name="sharepoint-hub-site-navigation"></a>SharePoint 中心网站导航

在中心网站顶部和任何关联网站 (网站导航上方) 是中心网站导航栏。 如果您是中心网站所有者, 则可以自定义[导航栏](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav)。 

**注意:** 建议选择一个通信网站, 或使用新式模板作为中心网站来选择一个团队网站。 如果使用经典团队网站, 中心导航将仅显示在新式页面上, 中心网站设置将仅显示在新式页面上。
