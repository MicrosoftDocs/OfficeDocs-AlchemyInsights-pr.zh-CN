# <a name="remote-work-setup-guide"></a>远程工作安装指南

远程执行的工作越来越多。 您的组织需要具备管理更多远程工作的需求。

使用 " [远程工作设置指南](https://go.microsoft.com/fwlink/?linkid=2142062) " 启用基本安全功能并查看其他安全建议。 你将获得指导以降低对你的 VPN 基础结构的压力，并可以选择为桌面和应用程序设置远程访问。 然后，你将查看用于部署 Microsoft 团队和 Office 应用的选项。
