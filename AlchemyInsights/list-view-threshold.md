# <a name="list-view-threshold"></a>列表视图阈值

SharePoint 列表和库可以包含多达 3 千万个项，但是当拥有的项超过 5,000 个并尝试使用它们时，可能会看到列表视图阈值错误。 此阈值是为了维持服务的性能。 不能更改。 尝试共享或中断文件夹的继承时，也可能会收到此错误。

为避免达到此阈值，请访问：

- [在 SharePoint 中管理大型列表和库](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59)

- [错误共享/中断继承](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
