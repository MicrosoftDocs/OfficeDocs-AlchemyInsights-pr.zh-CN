# <a name="help-understanding-value-added-tax-vat"></a>帮助了解增值税 (VAT)

如果您所在的国家/地区使用增值税 (VAT) (如英国), Microsoft 开票将为您计算 VAT, 并将金额应用于您的发票。
  
出于纳税原因, 您还必须在设置发票时指定 VAT 注册号。
  
您无法添加或删除 VAT 税率, 但您可以在 "设置" 中更改 "默认增值税率"。 默认情况下, 开票使用服务来验证 VAT 注册号码, 但我们建议您阅读欧洲委员会发布的[VIES 免责声明](https://go.microsoft.com/fwlink/?LinkID=841741)。 如果选择禁用该服务, 则必须验证所有 VAT 注册号码。
