# <a name="generate-automatic-captions-and-a-transcript-in-microsoft-stream"></a>在 Microsoft Stream 中生成自动字幕和记录

Stream 可以使用自动语音识别技术自动生成字幕，并且可以查看 "滚动记录" 窗口，以便在视频的讲述内容中轻松导航和跟踪。

- [生成 Microsoft Stream 视频的自动字幕和记录](https://docs.microsoft.com/stream/portal-autogenerate-captions)

- [在 Microsoft Stream 中显示视频的滚动脚本窗口](https://docs.microsoft.com/stream/portal-configure-transcript-mode)
