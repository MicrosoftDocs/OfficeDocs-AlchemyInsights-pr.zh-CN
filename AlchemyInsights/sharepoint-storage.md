# <a name="manage-your-sharepoint-online-storage"></a>管理 SharePoint Online 存储

对于大多数组织而言，因此您无需设置每个网站集的限制自动托管 SharePoint 存储。检查您有多少 SharePoint 存储：
  
- 在经典的 SharePoint 管理中心网站集页面的右上方查找可用并且总存储。
    
有关如何计算您限制的信息，请参阅[SharePoint Online 限制](https://go.microsoft.com/fwlink/p/?LinkID=856113)。若要购买额外的存储，请参阅[更改为您的订阅的存储空间](https://go.microsoft.com/fwlink/?linkid=866428)。
  
有关监视的各个网站存储信息使用 PowerShell 和管理网站集存储手动，请参阅[管理网站集存储限制](https://go.microsoft.com/fwlink/?linkid=867833)
