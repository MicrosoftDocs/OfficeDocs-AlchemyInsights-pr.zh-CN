# <a name="user-mailbox-creation-is-taking-a-long-time"></a>用户邮箱创建花费了很长时间

在新邮箱准备好使用前, 可能需要长达24小时。
  
如果用户的邮箱的 "正在准备" 的时间超过一小时, 请尝试删除用户的 Exchange 许可证并重新添加它, 或[与支持人员联系](https://docs.microsoft.com/office365/admin/contact-support-for-business-products?tabs=online)。
