# <a name="your-users-mailbox-is-being-created"></a>正在创建你的用户邮箱

在新邮箱准备好使用前, 可能需要长达24小时。
  
如果用户的邮箱的 "正在准备" 的时间超过一小时, 请尝试删除用户的 Exchange 许可证并将其重新添加回来, 或使用 "让我们呼叫你" 选项与支持工程师交谈。
