# <a name="issues-seeing-calendar-freebusy-information"></a>查看日历的忙/闲信息时出现的问题

如果在其他 Outlook 日历中看不到日历忙/闲信息，或者其他人无法查看日历中的忙/闲信息，请使用[适用于 Office 365 的 Microsoft 支持和恢复助理](https://diagnostics.office.com/)。
