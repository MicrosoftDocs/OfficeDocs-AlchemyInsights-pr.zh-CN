# <a name="create-a-classic-site"></a>创建经典网站

- 转到[Office 365 管理中心中的网站页面](https://portal.office.com/adminportal/home#/SitesList)上，然后选择**添加网站**。 
    
- 在经典 SharePoint 管理中心，在网站集页上，选择**新建** \> **专用网站集**。 
    
 **注意：** 无法创建新网站具有与现有网站相同的 URL。 
  
有关创建经典网站的详细信息，请参阅[创建网站集](https://go.microsoft.com/fwlink/?linkid=866295)。
  
有关允许用户创建自己的网站的信息，请参阅[SharePoint Online 中的管理网站创建](https://go.microsoft.com/fwlink/?linkid=866296)。
