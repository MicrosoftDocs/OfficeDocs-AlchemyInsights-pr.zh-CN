# <a name="change-the-font-size-in-a-message"></a>更改邮件中的字体大小

1. 如果已键入邮件，请选择该文本。
2. 选择邮件下方的 "  **格式** " 图标 (它是第一个) 的图标。
3. 选择邮件上方的 "  **字号**  " 图标 (它是第七个图标) 。
4. 选择一种大小。
