# <a name="delete-a-sharepoint-page"></a>删除 SharePoint 页面

1. 转到页上的站点。
    
2. 在左侧，单击**网站内容**。
    
3. 在网站内容页的左侧，单击**页面**。
    
4. 选择页，然后单击右侧的页上的省略号 （...）。
    
5. 单击“删除”。****
