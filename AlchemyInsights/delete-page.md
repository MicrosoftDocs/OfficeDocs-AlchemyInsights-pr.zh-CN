# <a name="delete-a-sharepoint-page"></a>删除 SharePoint 页面

1. 转到页面所在的网站。
    
2. 在左侧, 单击 "**网站内容**"。
    
3. 在 "网站内容" 页的左侧, 单击 "**页面**"。
    
4. 选择页面, 然后单击页面右侧的省略号 (...)。
    
5. 单击“删除”。****
