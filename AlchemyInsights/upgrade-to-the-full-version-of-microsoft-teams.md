# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>升级到 Microsoft 团队的完整版本

是否需要从 Microsoft 团队的完全版升级到完全版？ 请参阅[升级 Microsoft 团队免费订阅 microsoft 365 订阅版本](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

您需要满足以下要求：

- 你是注册当前免费版本的团队的人员。
- 如果你有自己的域，则它无法与 Azure Active Directory 相关联。
- 若要在升级时传输数据，您必须在升级过程中进行订阅。 如果您在升级过程之外购买了订阅，则无法传输数据。
