# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>在 web 上的 Outlook 中打开共享邮箱时是否获取权限错误？

如果遇到错误，则会出现错误 **。您没有打开此邮箱的权限。** 当您尝试在 web 上的 Outlook 中打开共享邮箱时，您的帐户可能无权访问共享邮箱。 您的组织管理员可以按照主题[创建共享邮箱](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)中的说明，将您的帐户权限分配给共享邮箱。
