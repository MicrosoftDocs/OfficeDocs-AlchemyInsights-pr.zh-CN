# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>如何通过 SharePoint 组自定义 SharePoint 网站权限 

SharePoint 组是对网站和内容具有相同权限集的用户的集合。 您可以使用组轻松地将相同的权限级别一次性分配给多个用户，而不是一次向一个人分配权限。

[自定义 SharePoint 网站权限](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
