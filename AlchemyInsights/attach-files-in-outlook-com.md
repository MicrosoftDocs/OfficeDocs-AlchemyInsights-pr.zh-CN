# <a name="how-to-attach-files"></a>如何附加文件

  
1. 撰写新邮件或答复邮件。
    
2. 在撰写窗格的底部, 选择回形针图标。
    
了解有关[如何在 Outlook.com 中附加文件的](https://go.microsoft.com/fwlink/p/?linkid=2001702&amp;clcid=0x409)详细信息。
