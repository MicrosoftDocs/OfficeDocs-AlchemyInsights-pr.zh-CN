# <a name="restore-deleted-sites"></a>还原已删除网站

删除的网站将保留 93 天。 93天后，将永久删除网站及其所有内容和设置，包括列表、库、页面和所有子网站。

- 有关更多信息，请参阅[还原已删除网站](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)。
