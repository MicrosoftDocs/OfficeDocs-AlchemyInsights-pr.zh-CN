# <a name="administrative-units"></a>管理单元

管理单元是 Azure AD 资源，可以是其他 Azure AD 资源的容器。 管理单元只能包含用户和组。

要了解有关管理单元的更多信息，请参阅 [Active Directory 中的管理单元](/azure/active-directory/roles/administrative-units)。
