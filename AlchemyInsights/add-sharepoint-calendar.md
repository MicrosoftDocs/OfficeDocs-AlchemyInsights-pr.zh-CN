# <a name="add-a-sharepoint-calendar"></a>添加 SharePoint 日历

SharePoint 日历是经典列表类型，有时会与 Microsoft 365 组日历混淆。
 
若要创建 SharePoint 日历，请按照以下步骤操作：
 
1.  选择"**设置"** 齿轮图标，然后选择"**添加应用"。**
2.  在" **查找应用搜索** "框中，键入 **"日历** "，然后按 Enter。
3.  选择标题为 **"日历"的应用**，然后输入日历的名称。

有关 Microsoft 365 组详细信息，请参阅了解 [Microsoft 365 组](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)。
