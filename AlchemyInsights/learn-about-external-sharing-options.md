# <a name="learn-about-external-calendar-sharing-options"></a>了解外部日历共享选项

如果需要安排与组织外部人员的会议, Office 365 使您能够为那些需要查看忙/闲时间但对 Office 365 环境没有用户帐户的 "外部用户" 可用的日历。 有关详细信息, 请参阅[与外部用户共享日历](https://support.office.com/article/fb00dd4e-2d5f-4e8d-8ff4-94b2cf002bdd.aspx)。
