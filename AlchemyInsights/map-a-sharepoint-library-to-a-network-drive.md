# <a name="map-a-sharepoint-library-to-a-network-drive"></a>将 SharePoint 库映射到网络驱动器

将库映射为网络驱动器是临时的，并且仅通过 Internet Explorer 支持。 相反，请[使用新的 OneDrive 同步客户端同步 SharePoint 文件](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx)，这将按[需提供文件](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx)。 在不使用本地存储空间的情况下访问 OneDrive 中的所有文件。
