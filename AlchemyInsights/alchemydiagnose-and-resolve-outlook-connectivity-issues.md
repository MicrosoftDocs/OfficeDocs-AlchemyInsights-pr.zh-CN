# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>诊断和解决 Outlook 连接问题

为了诊断和解决您的 Outlook 连接问题，可以在受影响的计算机上运行[Outlook 连接工具](https://aka.ms/SaRA-OutlookDisconnect)。 
  
此工具将执行诊断的检查和建议，用于解决 Outlook 连接问题可能的解决方案。
