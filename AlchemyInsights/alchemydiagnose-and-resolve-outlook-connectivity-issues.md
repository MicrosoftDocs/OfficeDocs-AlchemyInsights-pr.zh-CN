# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>诊断和解决 Outlook 连接问题

为了诊断和修复 outlook 连接问题, 您可以在受影响的计算机上运行[outlook 连接工具](https://aka.ms/SaRA-OutlookDisconnect-Alchemy)。
