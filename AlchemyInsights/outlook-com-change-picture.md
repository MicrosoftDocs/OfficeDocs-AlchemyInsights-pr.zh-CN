# <a name="change-your-account-picture"></a>更改您的用户头像

1. 转到[你的 Microsoft 帐户信息](https://go.microsoft.com/fwlink/p/?linkid=860841)。
2. 在图片下, 选择 "**更改图片**"。
3. 选择 "**新图片**"。
4. 选择要使用的图片, 然后选择 "**打开**"。
5. 拖动图像框或调整其大小, 然后选择 "**保存**"。

请允许最长24小时对你的帐户照片进行更改以生效。
