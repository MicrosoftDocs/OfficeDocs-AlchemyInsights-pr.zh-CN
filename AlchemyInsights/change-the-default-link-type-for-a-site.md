# <a name="change-the-default-link-type-for-a-site"></a>更改网站的默认链接类型

用户可以通过发送链接在 SharePoint 和 OneDrive 中共享文件和文件夹。 他们应该根据要授予权限的人员选择链接类型。

- [更改网站的默认链接类型](https://docs.microsoft.com/sharepoint/change-default-sharing-link)
