# <a name="unable-to-delete-photos-from-files"></a>无法从文件中删除照片

若要从 Outlook.com 中的文件中删除照片：
  
1. 转到 **"收件箱" 或 "** **已发送**邮件" 文件夹，并删除其附加的电子邮件。

2. 从**已删除的项目**中删除项目。

有关详细信息，请参阅[如何在 Outlook.com 中删除文件或照片](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)。
