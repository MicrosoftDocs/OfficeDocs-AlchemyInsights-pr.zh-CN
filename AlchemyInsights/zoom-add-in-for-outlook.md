# <a name="zoom-add-in-for-outlook"></a>Outlook Zoom 加载项

Zoom 计划程序加载项可让你轻松将 Zoom 会议添加到任何新的或现有的 Web 版 Outlook 日历事件上。

可在[此处](https://go.microsoft.com/fwlink/?linkid=2126413)下载加载项。
