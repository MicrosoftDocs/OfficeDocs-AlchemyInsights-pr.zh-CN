# <a name="export-user-account-information"></a>导出用户帐户信息

若要在 Microsoft 365 管理中心中导出用户帐户信息, 包括有关许可证、密码策略、阻止帐户、AAD 同步状态等的信息, 请转到 "**用户** > [活动用户](https://go.microsoft.com/fwlink/p/?linkid=834822)" 页, 然后选择 "导出"。 **用户**。
