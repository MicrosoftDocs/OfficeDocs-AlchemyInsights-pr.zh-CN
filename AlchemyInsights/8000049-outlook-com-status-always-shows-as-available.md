# <a name="my-outlookcom-status-always-shows-as-available"></a>我 Outlook.com 的状态始终显示为空闲

您的帐户状态显示您的帐户图片旁边，可用于您 Outlook.com 和 Skype 联系人知道您是否可用于聊天或具有 Skype 调用。您可以更改您的帐户状态 Outlook.com 或 Skype，但您登录到 Outlook.com，每次您的帐户状态将重置为有空。
  
您可以更改使用 Outlook.com beta，或不注销 Outlook.com 阻止您的状态。请注意，关闭浏览器不相同注销并不影响您的状态。关闭浏览器时，请保持登录，在提示符中选择**使我保持登录**框中的 at 符号。
