# <a name="close-your-outlookcom-email-account"></a>关闭 Outlook.com 电子邮件帐户

1. 转到[关闭您的帐户](https://go.microsoft.com/fwlink/p/?linkid=845493)页上。 
    
2. 按照以确保您的帐户已准备好关闭，页上的说明，然后选择**下一步**。 
    
3. 读取列表和复选框，以确认您已经阅读每个项目。
    
4. 在**选择原因**下拉列表中，选择您正在结束帐户的原因。 
    
5. 选择**关闭标记帐户**。 
    
若要了解详细信息，请参阅[关闭 Outlook.com 帐户](https://go.microsoft.com/fwlink/p/?linkid=873106)[](https://support.office.com/article/564b801e-2a47-4cb2-afa8-12ead3185038.aspx)。
