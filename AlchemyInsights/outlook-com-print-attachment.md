# <a name="print-an-attachment"></a>打印附件

1. 打开包含要打印的附件的邮件。
2. 将光标移到附件上, 然后选择 "**下载**" 图标。
3. 下载文件后, 使用相应的程序将其打开, 然后选择要打印的选项。 例如, 在 Word 中打开一个 .doc 文件, 然后打印。

有关详细信息, 请参阅[Outlook.com 中的 "打印电子邮件"、"附件" 和 "日历](https://go.microsoft.com/fwlink/?linkid=2021110&amp;clcid=0x409)"。
