# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>在 Mac 上安装、设置和同步 OneDrive 或 SharePoint 文件 

使用[MAC OS X 上的 onedrive 同步客户端同步文件](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f)中的说明, 在 mac 上安装、设置和同步 Onedrive 或 SharePoint 文件。

如果已在 Mac 上安装和设置 OneDrive 应用, 但用户遇到同步问题, 请参阅[修复与 OneDrive (Mac) 的同步问题](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US)。

> [!NOTE]
> OneDrive sync for Windows 中的已知文件夹移动功能对 Mac 不可用。
