# <a name="reactivate-a-microsoft-365-subscription"></a>重新激活Microsoft 365订阅

如果你的订阅过期、被 Microsoft 禁用或在订阅期中取消了订阅，可以在管理中心重新激活它。
  
1. 在管理中心中，转到“**账单**” > “[你的产品](https://go.microsoft.com/fwlink/p/?linkid=842054)”页面。
2. 在“**产品**”选项卡上，找到要重新激活的订阅。 Select the three dots (more actions) ， then select **Reactivate this subscription**.
3. 在“**重新激活此订阅**”窗格中，从下拉列表中选择支付方式，或选择“**添加付款方式**”。
4. 选择“**保存**”。

要了解详细信息，请参阅[重新激活订阅](/microsoft-365/commerce/subscriptions/reactivate-your-subscription)。
