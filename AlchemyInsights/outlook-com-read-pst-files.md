# <a name="pst-data-files-in-outlookcom"></a>Outlook.com 中的 PST 数据文件

不能导入 Outlook。PST 数据文件使用浏览器向您的 Outlook.com 帐户。

若要导入。PST 数据文件, 则必须使用 Outlook 桌面应用程序。 若要了解详细信息, 请参阅[从 Outlook .Pst 导入电子邮件、联系人和日历。](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
