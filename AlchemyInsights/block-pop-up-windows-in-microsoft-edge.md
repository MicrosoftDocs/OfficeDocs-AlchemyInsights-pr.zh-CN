# <a name="block-pop-up-windows-in-microsoft-edge"></a>在 Microsoft Edge 中阻止弹出窗口

1. 转到... **(设置和更多)**  >  **设置**  >  **网站权限**。
2. 选择 **"弹出窗口和重定向"。**
3. 打开阻止。
