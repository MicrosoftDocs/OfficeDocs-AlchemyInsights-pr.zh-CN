# <a name="restore-deleted-files-or-folders-in-onedrive"></a>在 OneDrive 中还原已删除的文件或文件夹

如果您意外删除了 OneDrive 中的某个文件或文件夹，可以稍后从 OneDrive 回收站恢复它。

* [在 OneDrive 中还原已删除的文件或文件夹](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

如果大量 OneDrive 文件被恶意软件删除、覆盖、损坏或感染，则可以将整个 OneDrive 还原到以前的时间。

* [还原你的 OneDrive](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
