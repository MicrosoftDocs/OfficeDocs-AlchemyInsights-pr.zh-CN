# <a name="increase-the-size-of-a-shared-mailbox"></a>增加共享邮箱的大小

您可以通过为共享邮箱分配 Exchange Online P2 许可证（它包含在 E3 和 E5 许可证中），将共享邮箱的大小从 50 GB 的默认大小增加到 100 GB。 将 P2 许可证分配给[活动用户](https://portal.office.com/adminportal/home)中的共享邮箱时，大小将在15分钟内自动增加。
