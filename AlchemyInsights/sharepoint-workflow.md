# <a name="workflows-in-sharepoint"></a>SharePoint 中的工作流

SharePoint 工作流是预先编程的应用程序，可简化多种业务过程。 有关详细信息，请参阅下面的链接。

## <a name="getting-started"></a>入门

如果只是开始使用 SharePoint 工作流，请参阅[简介到 sharepoint 工作流](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3)。

## <a name="troubleshoot-and-debug-a-sharepoint-workflow"></a>SharePoint 工作流疑难解答和调试

请参阅[sharepoint 工作流验证错误](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio)和[调试 Sharepoint 工作流](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows)疑难解答。

## <a name="power-automate"></a>电源自动化

请考虑使用[Power 自动执行](https://docs.microsoft.com/power-automate/modern-approvals)，而不是经典 SharePoint 工作流解决方案。
