# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>了解 Microsoft Edge 如何支持 (WIP) 的 Windows 信息保护

随着远程工作人员数量的增加，在企业设备上发生的个人和工作活动的数量也会增加。 这反过来又增加了公司数据可能会在工作场所之外共享的风险。 若要缓解此风险，Microsoft Edge (版本81或更高版本) 支持 Windows 信息保护 (WIP) ：一种帮助保护企业数据免遭未经授权和意外披露的 Windows-10 功能。
