# <a name="update-or-add-a-new-payment-method"></a>更新或添加新的付款方式

1. 转到 "**计费** > **帐单 & 付款** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">方法</a>" 页。
2. 选择 "**添加付款方式**" 或选择 "要更新的信用卡或银行帐户"，然后选择 "**编辑**"。
3. 更新付款方式信息（名称在卡片或银行帐户、帐单地址、电话号码或到期日期），然后选择 "**保存**"。
4. 请记住更改你将订阅付费的方式添加到此新的付款方式。 有关如何执行此操作的说明，请在下一节中进行操作。

如果你已添加新卡，但仍收到有关旧信用卡的拒绝通知，则可能需要将该新卡与订阅关联。

1. 转到**帐单** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">产品 & 服务</a>
2. 选择要使用新的卡或银行帐户支付的订阅。 在 "**记帐**" 部分下，选择 "付款方法" 旁边的 "**编辑**"。
3. 选择当前付款方式旁边的 "**更改**"。
4. 从下拉菜单中，选择另一种付款方式，或添加新的支付方式。

[添加、更新或删除支付方式](https://go.microsoft.com/fwlink/?linkid=2118133)
