# <a name="live-events-in-microsoft-stream"></a>Microsoft Stream 中的实时事件

您可以使用 Microsoft Stream 在整个组织中创建实时事件。 您可以为各种方案（如公司范围的事件、领导更新等）安排、生成和传递活动事件。

- [Microsoft Stream 中的实时事件](https://docs.microsoft.com/stream/live-event-overview)
- [在 Microsoft Stream 中创建实时事件](https://docs.microsoft.com/stream/live-create-event)
- [Microsoft 团队 live 事件入门](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [在 Microsoft Stream 中配置实时流的编码器](https://docs.microsoft.com/stream/live-encoder-setup)
