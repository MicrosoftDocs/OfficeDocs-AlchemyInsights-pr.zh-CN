# <a name="cant-delete-files-or-photos-from-files"></a>无法从文件中删除文件或照片

**文件**显示附加到邮箱中邮件的文件和照片。 若要从文件中删除文件和照片, 您需要删除其附加的电子邮件。 删除邮件后, 还需要将其从 "已删除邮件" 文件夹中删除。
