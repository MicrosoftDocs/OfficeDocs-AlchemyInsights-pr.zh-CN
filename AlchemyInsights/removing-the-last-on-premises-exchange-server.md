# <a name="removing-the-last-on-premises-exchange-server"></a>删除最后一个内部部署 Exchange server

从内部部署组织删除最后一个 Exchange server 之前, 您应该确保您了解此操作此选项可帮助您决定是否需要的最后一台服务器的任何其他目的的含义。有关详细信息，请查看[如何以及何时停用混合部署中的本地 Exchange 服务器](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx)。
