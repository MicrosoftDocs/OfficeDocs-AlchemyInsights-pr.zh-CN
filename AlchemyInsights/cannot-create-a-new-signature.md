# <a name="cannot-add-or-create-a-new-signature-in-outlook"></a>无法在 Outlook 中添加或创建新签名

这是一个已知问题。 我们正在努力解决该问题，Outlook 中的状态将会在 Outlook 中更新[冻结单击 "签名" 按钮](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)。

若要解决此问题，请先尝试卸载预安装的 Microsoft Office 桌面应用程序的[步骤](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)。 

如果这对受影响的计算机不起作用，请尝试使用[基于注册表的替代方法](https://support.office.com/article/c70b36c2-66ca-401c-ab45-f29a46495d02)。
