# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>将通讯组转换为 Microsoft 365 组

您可以将通讯组转换为 Microsoft 365 组（如果它们满足特定条件）。 若要转换组或解决转换问题，请参阅[Outlook 中的将通讯组列表升级到 Microsoft 365 组](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)。
