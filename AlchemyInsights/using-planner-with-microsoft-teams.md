# <a name="using-planner-with-microsoft-teams"></a>将 Planner 与 Microsoft Teams 一同使用

通过将一个或多个 Planner 选项卡添加到 Teams 频道来组织任务： 

- [将 Planner 选项卡添加到团队频道](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_addaplannertabtoateamchannel)。 添加选项卡的用户必须是团队的成员，而不只是所有者。 可以通过 Microsoft Teams 界面将用户添加为成员。
- [将现有计划添加到 Microsoft Teams](https://techcommunity.microsoft.com/t5/Planner-Blog/Bringing-a-Plan-into-Microsoft-Teams/ba-p/57463)。
- [删除 Planner 选项卡或删除计划](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_removeaplannertabordeleteaplan)。 无法在 Teams 中恢复已删除的计划。 如果已删除组，可以恢复 [已删除的计划和组](https://blogs.msdn.microsoft.com/brismith/2017/03/29/microsoft-planner-now-you-can-recover-deleted-plans-and-groups)。
- [在 Teams 中启用或禁用 Planner 分配通知](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_getplannerassignmentnotificationsinteams)。

**注意：** 有关 Planner 与 Microsoft Teams 集成的一般概述，请参阅在 [Microsoft Teams 中使用 Planner。](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0)
