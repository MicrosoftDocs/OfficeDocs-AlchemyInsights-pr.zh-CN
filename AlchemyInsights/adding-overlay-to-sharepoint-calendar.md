# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>向 SharePoint 日历添加覆盖

SharePoint 日历是经典列表类型，有时会与 Microsoft 365 组日历混淆。
 
如果网站上使用 SharePoint 日历，可以将自己的 Exchange 日历覆盖到该日历上，以创建一个合并的日历视图。 或者，只要两个 SharePoint 日历都在同一个网站集中，就可以覆盖另一个 SharePoint 日历。
 
若要向经典 SharePoint 日历添加覆盖，请按照使用 Exchange 或 SharePoint 中的日历覆盖 [SharePoint 日历中的步骤操作](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A)。
