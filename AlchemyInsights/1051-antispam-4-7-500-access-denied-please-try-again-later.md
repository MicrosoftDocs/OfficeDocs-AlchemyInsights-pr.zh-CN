# <a name="47500-access-denied-please-try-again-later"></a>4.7.500 访问被拒绝，请稍后重试

来自源 IP 地址的电子邮件通信量超出限制基于源 IP 地址的信誉 （或缺乏信誉） 时，将发生此错误。
  
一个小时内过期阻止来自源 IP 地址的电子邮件。如果源 IP 地址是属于您的内部部署电子邮件服务器，请验证邮件流连接器的配置。如果行为依然存在多个小时，联系支持人员请求源 IP 地址的例外。
