# <a name="add-a-guest-to-microsoft-teams"></a>将来宾添加到 Microsoft 团队

有时，您可能希望与工作组组织中不存在的人员进行协作。 通常会在电子邮件中执行此操作，但如果您想要在团队中与他们进行协作，则可以将其添加为来宾。

有关详细信息，请参阅[向团队中的团队添加来宾](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop)。

**注意：** 您必须是团队所有者才能在团队中添加来宾（另请参阅 "[设置来宾权限](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)"）。
