# <a name="changing-ews-throttling-settings"></a>更改 EWS 限制设置

请运行我们的自动测试，该测试将允许你修改针对迁移持续时间的 EWS 限制策略。

请注意，EWS 限制策略更改对以下迁移类型没有影响（使用 Microsoft 工具）：混合、直接转换/暂存（RPC/HTTP）、IMAP、G 套件、公共文件夹或 PST 导入服务。
