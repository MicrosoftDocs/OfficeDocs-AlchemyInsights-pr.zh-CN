# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>从 SharePoint 回收站还原文件或文件夹 

当您从 SharePoint 文档库或列表中删除某个项目时，不会立即将其从 SharePoint 中删除。 已删除的项目会进入网站回收站一段时间。 在此期间，可以将已删除的项目还原到原始位置。

- [从 SharePoint 回收站还原文件或文件夹](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be)
- [在 OneDrive 中还原已删除的文件或文件夹](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)
