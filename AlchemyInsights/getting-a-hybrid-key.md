# <a name="getting-an-exchange-hybrid-key"></a>获取 Exchange 混合密钥

要连接到 Office 365 中 Exchange Online 的内部部署组织，您需要设置内部部署组织中的至少一个 Exchange 服务器。此服务器用于创建连接到 Exchange Online 的内部部署 Exchange 组织的混合部署。要为此服务器获取混合密钥，请按照[Exchange 混合产品密钥分发向导](http://aka.ms/hybridkey)的步骤。
