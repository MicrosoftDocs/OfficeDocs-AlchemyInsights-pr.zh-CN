# <a name="getting-an-exchange-hybrid-key"></a>获取 Exchange 混合密钥

若要将内部部署组织连接到 Office 365 中的 exchange Online, 您需要在内部部署组织中设置至少一个 exchange 服务器。 此服务器用于创建将内部部署 Exchange 组织连接到 Exchange Online 的混合部署。 若要获取此服务器的混合密钥, 请按照[Exchange 混合产品密钥分发向导](http://aka.ms/hybridkey)中的步骤操作。
