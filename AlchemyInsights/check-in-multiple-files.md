# <a name="check-in-several-files-at-once"></a>一次签入多个文件

签入和签出文件是跟踪和控制内容更新方式的一种非常好的方法。 如果要对网站上的文件进行更改, 并希望确保没有人可以对其进行编辑, 请签出该文件。 签出文件后, 可以联机或脱机编辑该文件, 并在必要时将其保存多次。

以下是有关此功能的一些有用链接。

- [关于签出、签入和版本的热门问题](https://support.office.com/article/Top-questions-about-check-out-check-in-and-versions-7E941339-E972-4C7A-A79A-80A1FCF84076)

- [一次签入多个文件](https://support.office.com/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de)
