# <a name="sharepoint-new-features-announced"></a>已发布的 SharePoint 新功能

在 Ignite 2018 会议中，Microsoft 宣布了新的 SharePoint 创新，包括新式门户的新功能、新的 web 部件、对中心网站、新闻链接、访问群体设定的大量改进、新移动应用的可用性、与团队的连接，以及更多 .。。
  
有关这些功能的详细信息，请参阅博客文章 [SharePoint 中新 &amp; 的创新 Microsoft 365 提供了内容协作的强大功能和简洁性。](https://go.microsoft.com/fwlink/?linkid=2026502)
  
您的组织可以加入目标发布程序以获得对新功能的早期访问权限。 请参阅 [在 Microsoft 365 中设置标准或目标发布选项](https://docs.microsoft.com/microsoft-365/admin/manage/release-options-in-office-365)。
