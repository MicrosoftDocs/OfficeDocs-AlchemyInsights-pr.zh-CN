# <a name="microsoft-365-multi-geo"></a>Microsoft 365 多地理位置

使用 Microsoft 365 多地理位置，你的组织可以将其 Microsoft 365 状态扩展到现有租户中的多个地理区域和/或国家/地区。 有关详细信息，请参阅下面的链接。

- [Microsoft 365 多地理位置](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [OneDrive 和 SharePoint Online 中的多地理位置功能](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-onedrive-and-sharepoint-online-in-office-365)
- [Exchange Online 中的多地理位置功能](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-exchange-online)
- [规划 Microsoft 365 多地理位置](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [配置多地理位置](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
