# <a name="power-bi-report-server-product-key"></a>Power BI 报表服务器产品密钥

如果已购买 Power BI Premium，可以在 Power BI 管理门户的"高级设置"选项卡中访问 Power BI 报表服务器产品密钥。 这仅适用于全局管理员或分配了 Power BI 服务管理员角色的用户。

选择 **Power BI 报表服务器密钥** 将显示一个包含产品密钥的对话框。 你可以将其复制并用于安装。

如果你有企业版 SA SQL Server，可以从批量许可服务中心获取 [产品密钥](https://www.microsoft.com/Licensing/servicecenter/)。

有关详细信息： https://docs.microsoft.com/power-bi/report-server/install-report-server
