# <a name="question-about-licenses"></a>有关许可证问题

如果您想要知道您已购买的许可证数量或从订阅分配多少许可证：
  
1. 在管理中心，转到[订阅](https://go.microsoft.com/fwlink/p/?linkid=842054)页面，或选择" **帐单**"\>" **订阅**"。
    
2. 选择您想要了解更多相关的订阅。
    
3. 在左侧的订阅卡是有关许可证的信息：
    
  - **购买**数是您支付的许可证数量。 
    
  - **已分配**数是分配给用户的许可证数量。 
    
4. 如果您想要添加更多许可证，或如果您有太多，选择要调整购买量**添加/删除许可证**。
