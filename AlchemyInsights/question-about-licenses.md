# <a name="questions-about-your-microsoft-365-license"></a>有关你的 Microsoft 365 许可证的问题

按照这些步骤确定购买了多少个许可证，或从订阅中分配了多少个许可证。
  
1. 在管理中心中，转到“**计费**”\>“[你的产品](https://go.microsoft.com/fwlink/p/?linkid=842054)”页面。
2. 在 " **产品** " 选项卡上，找到要详细了解的订阅。 您可以查看要支付的许可证数量以及当前分配的许可证数量。

如果要添加更多许可证，或者如果您具有过多许可证，请选择订阅。 在 "订阅数据" 页上，选择 " **购买许可证** " 或 " **删除许可证** " 以调整订阅的许可证总数。
