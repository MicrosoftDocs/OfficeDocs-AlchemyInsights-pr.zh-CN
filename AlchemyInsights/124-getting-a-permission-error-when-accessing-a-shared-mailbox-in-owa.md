# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>在 web 上的 Outlook 中打开共享的邮箱时，请获取权限错误？

如果出现错误**出现问题。您没有打开此邮箱的权限。** 当您尝试在 web 上的 Outlook 中打开共享的邮箱时，您的帐户可能没有共享的邮箱的访问权。您组织的管理员可以将帐户权限分配给共享邮箱中，按照主题[创建共享邮箱](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9)中的说明。
