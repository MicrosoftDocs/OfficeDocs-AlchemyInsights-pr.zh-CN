# <a name="manage-synchronized-user-accounts"></a>管理同步的用户帐户

通常，您可以使用 Office 365 管理中心添加和更改用户帐户。但是，如果则无法使用它启用目录同步还是 SSO。相反，管理您的用户使用 Windows PowerShell[配置用户帐户属性与 Office 365 PowerShell](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell )中所述。
