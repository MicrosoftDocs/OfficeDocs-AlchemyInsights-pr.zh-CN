# <a name="help-enabling-school-data-sync"></a>帮助启用学校数据同步

需要帮助以启用学校数据同步？ 你知道你可以从 Microsoft 365 FastTrack 程序获取帮助吗？

请访问 [FastTrack 中心](https://www.microsoft.com/fasttrack)，提交帮助请求，为组织启用学校数据同步。
