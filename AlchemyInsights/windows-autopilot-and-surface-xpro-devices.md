# <a name="windows-autopilot-and-surface-x-pro-devices"></a>Windows Autopilot 和 Surface X Pro 设备

Windows Autopilot 是 Windows 10 中的一种基于云的部署技术。 可以使用 Windows Autopilot 立即远程部署和配置设备。 Windows Autopilot 注册的设备在首次启动时通过称为硬件哈希的唯一设备签名通过 Internet 进行标识。 它们使用现代管理解决方案进行配置，例如 Azure Active Directory (Azure AD) 和 Microsoft Intune。

设备可以直接交付给用户。 此过程在部署过程中消除了重新映像，从而允许你实现新的敏捷的设备管理和分发方法。

若要了解更多信息，请参阅 [Windows Autopilot 和 Surface 设备](https://go.microsoft.com/fwlink/?linkid=2135712)。
