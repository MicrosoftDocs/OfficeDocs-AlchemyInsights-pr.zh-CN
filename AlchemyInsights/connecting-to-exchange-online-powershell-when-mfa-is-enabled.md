# <a name="connect-to-exchange-online-powershell-when-mfa-is-enabled"></a>启用 MFA 时连接到 Exchange Online PowerShell

如果你的帐户启用了多重身份验证（MFA），则需要按照以下说明连接到 Exchange Online PowerShell：[使用 Exchange Online powershell with 新式 authentication Using V2 module](https://aka.ms/exops-docs)。 请在该页面上搜索 MFA/多重因素身份验证说明。
