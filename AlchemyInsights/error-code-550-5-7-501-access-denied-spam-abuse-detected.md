# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 访问被拒绝, 检测到的垃圾邮件滥用

通常, 当用户使用分配给 Office 365 中新租户的*onmicrosoft.com*域从 IP 地址发送电子邮件时, 将会出现此消息。 解决此问题最简单的方法是:

1. [将域添加到你的租户](https://support.office.com/article/6383f56d-3d09-4dcb-9b41-b5f5a5efd611.aspx)。

2. [将用户的主电子邮件地址更改](https://support.office.com/article/fb5ac074-e203-4e1f-9843-b9d1a3e03297.aspx)为刚添加的新自定义域。
