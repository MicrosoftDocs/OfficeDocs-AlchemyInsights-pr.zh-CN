# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>访问被拒绝的 550 5.7.708，通信不接受来自此 IP

从已知、 低信誉通常由新客户使用的 IP 地址发送电子邮件时，将发生此错误。
  
如果之前可以购买许可证，您必须从这些 IP 地址发送电子邮件，请联系支持以请求异常，直到您能购买许可证。
