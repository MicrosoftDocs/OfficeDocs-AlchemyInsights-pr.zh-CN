# <a name="limit-sharing-in-microsoft-365"></a>限制 Microsoft 365 中的共享

虽然不能完全禁用内部共享，也不能从网站中删除 "共享" 按钮，但有多种方法可以限制 Microsoft 365 中的共享以满足组织的需求。 有关详细信息，请参阅 [限制在 Microsoft 365 中共享](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)。
