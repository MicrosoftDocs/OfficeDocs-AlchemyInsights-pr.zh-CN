# <a name="limit-sharing-in-microsoft-365"></a>限制 Microsoft 365 中的共享

虽然无法完全禁用内部共享，或者无法从网站中删除"共享"按钮，但有多种方法可以限制 Microsoft 365 中的共享以满足组织的需要。 有关详细信息，请参阅 Microsoft [365 中的限制共享](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing)。
