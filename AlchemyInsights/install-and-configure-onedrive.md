# <a name="install-and-configure-onedrive"></a>安装和配置 OneDrive

OneDrive 同步客户端是在 Windows 10 中预安装的。 若要安装 OneDrive for Windows 7 或 Mac, 或者为你的手机或平板电脑获取移动应用, 请[http://onedrive.com/download](http://onedrive.com/download)转到。
  
若要在 Windows 中设置 OneDrive 同步客户端, 请参阅:
  
- [将计算机设置为同步 Office 365 中的 OneDrive for Business 文件](https://go.microsoft.com/fwlink/?linkid=533375)

- [通过新的 OneDrive 同步客户端同步 SharePoint 文件](https://go.microsoft.com/fwlink/?linkid=871666)
