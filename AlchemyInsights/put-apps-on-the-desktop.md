# <a name="put-apps-on-the-desktop"></a>将应用放在桌面上

在 "**开始**" 菜单中搜索应用程序，并通过按住鼠标左键并将图标移动到桌面来获取应用程序图标。

您还可以按**Windows 键 + R** ，然后键入以下命令：

`shell:appsfolder`

这将在**文件资源管理器**中打开 "**应用程序**" 文件夹。 您现在可以将应用程序图标拖放到桌面上。
