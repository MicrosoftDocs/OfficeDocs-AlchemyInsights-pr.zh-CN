# <a name="put-apps-on-the-desktop"></a>将应用置于桌面上

在"开始"菜单中搜索应用，然后按住鼠标左键，将图标移动到桌面，获取应用图标。

还可以按 Windows **键 + R，** 然后在此命令中键入：

`shell:appsfolder`

这将在文件 **资源管理器中** 打开应用程序 **文件夹**。 现在，你可以将应用图标拖放到桌面。
