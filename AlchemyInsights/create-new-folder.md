# <a name="create-a-new-folder"></a>新建文件夹

1. 在导航窗格中的 "**文件夹**" 下, 选择 "**新建文件夹**"。 
    
2. 为该文件夹键入一个名称, 然后按 enter。
    
若要了解详细信息, 请参阅[在 Outlook.com 中创建文件夹](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)。
