# <a name="set-passwords-to-never-expire"></a>将密码设置为永不过期

将密码设置为永不过期[安全&amp;隐私](https://portal.office.com/adminportal/home#/settings/security)。 单击 "密码策略" 旁边的 "编辑", 然后将滑块转到 "打开"。
  
有关更多详细信息, 请参阅: 为[组织设置密码过期策略](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f)。
