# <a name="set-passwords-to-never-expire"></a>设置密码永不过期

设置密码永不过期从[安全&amp;隐私](https://portal.office.com/adminportal/home#/settings/security)。旁边密码策略，单击"编辑"，然后为"打开"将滑块。
  
有关详细信息，请参阅：[设置为您的组织的密码到期策略](https://support.office.com/en-us/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f)。
