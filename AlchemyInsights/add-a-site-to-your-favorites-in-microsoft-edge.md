# <a name="add-a-site-to-your-favorites-in-microsoft-edge"></a>在 Microsoft Edge 中将网站添加到收藏夹

1. 在 Microsoft Edge 中，转到要添加的站点。
2. 在地址栏中，选择" **将此页添加到收藏夹"** 图标。 或者，选择 **Ctrl+D。**
3. 可选：在 **"编辑收藏夹** "对话框中，重命名收藏夹并选择另一个文件夹将其保存。
4. 然后，在" **编辑** 收藏夹 **"对话框中选择"完成** "。
