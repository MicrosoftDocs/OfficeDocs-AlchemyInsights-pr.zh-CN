# <a name="create-email-signatures"></a>创建电子邮件签名

这些主题将帮助您创建电子邮件签名：
  
- 创建您自己的邮件的电子邮件签名：[创建并向消息中添加签名](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)或[创建和 Outlook Web App 中添加电子邮件签名](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx)。
    
- 创建从您的公司 （也称为免责声明） 发送的所有邮件的全局电子邮件签名：[组织范围内邮件免责声明、 签名、 页脚或 Office 365 中的邮件头](https://go.microsoft.com/fwlink/p/?linkid=391096)。
