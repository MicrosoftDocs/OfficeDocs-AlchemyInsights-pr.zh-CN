# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>为 Microsoft Edge 创建终结点的允许列表

若要在您的工作中拥有完整的 Microsoft Edge 功能阵列，您必须将其连接到 Internet。 此外，若要让浏览器通过 Internet 上的防火墙和其他安全机制进行通信，则必须将一组域 Url 添加到其 [允许终结点列表](https://go.microsoft.com/fwlink/?linkid=2135054)中。
